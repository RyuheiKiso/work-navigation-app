| [01_管理](./01_管理/README.md) | 構成管理・変更管理・規約 | 支援プロセス |

| [05_詳細設計](./05_詳細設計/README.md) | モジュール・DB・画面・処理の詳細 | ソフトウェア詳細設計 |
| [06_実装](./06_実装/README.md) | 実装メモ・ビルド/デプロイ手順 | ソフトウェア構築 |
| [07_テスト](./07_テスト/README.md) | テスト計画・仕様書・結果 | テストプロセス群 |
| [08_移行](./08_移行/README.md) | 移行計画・導入手順 | 導入プロセス |
