# システム化計画

IPA 共通フレーム 2013 (SLCP-JCF2013) に準拠した企画プロセス成果物。  
本ディレクトリは、要件定義・概要設計フェーズの **入力前提文書** として機能する。

## 文書目的

本計画書は「何を・なぜ・どのように作るか、スコープはどこまでか、誰のために、どんな制約と規制のもとで」を確定する。後工程での手戻りを防ぎ、8 種のステークホルダー全員が同一の前提を共有することを目的とする。

## 引用記法

本文書群での業界分析引用は以下の形式を使用する。

```
[`90_業界分析/XX_テーマ名.md`](../../90_業界分析/XX_テーマ名.md)
```

例: [`90_業界分析/22_規制別トレーサビリティ要件詳論.md`](../../90_業界分析/22_規制別トレーサビリティ要件詳論.md)

ASCII 図の使用は禁止。図は `img/` 配下の SVG ファイルを参照すること（[`docs/CLAUDE.md`](../../CLAUDE.md) 参照）。

規制に関する記述は「準拠する／対応する（拡張可能性確保）／対象外と判断する」の 3 段階表現のみ使用する。
