## 2. 管理コンソールのナビゲーション設計

管理コンソールはサイドバーナビゲーション方式を採用する。全画面で SCR-MC-001（ダッシュボード）への戻りが常時可能。

```
┌─────────────────────────────────────┐
│ [サイドバー]          [メインエリア] │
│                                      │
│ ダッシュボード  → SCR-MC-001        │
│ ユーザー管理    → SCR-MC-002        │
│ ロール管理      → SCR-MC-003        │
│ 監査ログ        → SCR-MC-004/005    │
│ バックアップ    → SCR-MC-006        │
│ DLQ 監視       → SCR-MC-007        │
│ ハッシュ検証    → SCR-MC-008        │
│ 帳票出力        → SCR-MC-009        │
└─────────────────────────────────────┘
```

---

**本節で確定した方針**
- **管理コンソールの TRN-041〜051 を確定し、9 画面（SCR-MC）のサイドバーナビゲーション方式を確定した。**
- **全遷移に RBAC ガード（system_admin/quality_admin）を設定し、executive ロールはダッシュボード（SCR-MC-001）のみアクセス可能とした。**
