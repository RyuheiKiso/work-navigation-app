# 07 ワイヤーフレーム（ハンディ APP・15 画面分）

本章の責務は、ハンディ APP の全 15 画面（SCR-HA-001〜015）について、コンポーネント配置・レイアウト領域・主要 UI 要素・インタラクションを確定することである。IPA 共通フレーム 2013「2.4 ヒューマンインタフェース設計」を担保し、NFR-UX 全 43 件のうち画面構造・タッチ操作・認知負荷に係る要件を設計として担保する。すべての図は `docs/CLAUDE.md` の規約に従い drawio 形式で作成した SVG ファイルを `img/` 配下に配置して参照する。

---

## 1. 共通レイアウトフレーム（360×800dp 標準）

高さ比率（%）指定を廃止し、8pt グリッド基準の絶対サイズと `flex` 一領域で定義する。

| 領域 | 高さ（dp）| 備考 |
|---|---|---|
| StatusBar / SafeArea（上部）| 36dp | OS 管理領域（固定）|
| OfflineBanner（CMP-HA-006）| 32dp | DISCONNECTED 以上のみ表示（条件付き）|
| **Header** | **56dp** | 常設。左 44dp タップ域（戻る/補助）、中央タイトル `text.h2`（20pt 600）、右 44dp（補助アクション）|
| **ProgressBar**（CMP-HA-007）| **36dp** | Step 実行画面（SCR-HA-004/005/006/007）のみ表示 |
| **主コンテンツ領域** | **flex（残余）** | 全スクロール可能 |
| **FooterCTA** | **88dp** | CTA ボタン 64dp + 上下余白 12/12dp。HomeIndicator 16dp は FooterCTA 内に吸収 |

**標準フレーム高さ計算**（全要素表示時）:
`36 + 32 + 56 + 36 + flex + 88 = 248 + flex`（800dp 端末では `flex = 552dp`）

**CTA ボタン仕様統一**:
- 幅: `container - 32dp`（左右 16dp パディング）= **328dp**
- 高さ: 64dp
- 角丸: `radius.md`（8px）
- フォント: `text.h3`（18pt / 700）
- primary: `brand.primary.500` 背景 / white テキスト

**AndonButton（CMP-HA-008）**:
- 位置: FooterCTA 領域外 / Floating / `bottom: 16dp, right: 16dp`
- サイズ: 72dp 円形（`radius.full`）
- `shadow.2` で FooterCTA とのレイヤー分離
- FooterCTA CTA との最小物理距離: 24dp（`06_` § 6-1 三角配置禁止規定）

**Header 仕様統一**:
- 背景: `surface.raised`（`shadow.3` で Sticky 時のレイヤー感）
- タイトル: 中央 `text.h2` 20pt / 600
- 戻るボタン: 左端 `ChevronLeft` icon.lg / tap area 44dp
- 補助アクション: 右端 44dp（任意）

**注意事項フォントルール（全画面必須）**:
- 警告・注意事項は **`text.body.lg`（16pt）以上 + `state.warning.50` 背景 + 左 4px `state.warning.500` ボーダー + `AlertTriangle` アイコン + weight 600**
- `text.caption`（12pt）の警告・エラー用途は全廃

- **color.{primary/success/warning/danger/andon} を新トークン体系（brand.primary.500 / state.success.500 等）に全面移行した。**
