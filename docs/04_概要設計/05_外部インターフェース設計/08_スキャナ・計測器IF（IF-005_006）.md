# 08 スキャナ・計測器 IF（IF-005/006）

本章の責務は、バーコード/QR スキャナ（IF-005）と IoT 計測器（IF-006）との接続方式を確定することである。いずれも端末（ハンディ APP）からの接続であり、バックエンドを経由しない。

---

## 1. スキャナ IF（IF-005）

### 1-1. 接続方式

| 接続種別 | 対応 | 優先度 |
|---|---|---|
| Bluetooth HID（BLE）| Android/iOS/Windows 対応 | 推奨 |
| USB HID | Android（OTG）/ Windows 対応 | 代替 |
| 組み込みカメラ（ソフトウェアデコード）| 全プラットフォーム | フォールバック |

### 1-2. GS1 バーコード解釈（BR-BUS-034）

```typescript
// 受信したバーコードデータの解釈
import { GS1Parser } from '@gs1/gs1-js';

function parseBarcode(rawData: string): Gs1Data {
  const parser = new GS1Parser();
  const result = parser.parse(rawData);
  
  if (!result.isValid()) {
    throw new DomainError('ERR-VAL-003', 'GS1形式が不正です');
  }
  
  return {
    lot_id: result.ai['10'],        // GS1 AI 10 = Lot/Batch Number
    product_code: result.ai['01'], // GS1 AI 01 = GTIN
    ...
  };
}
```

### 1-3. フォールバック

スキャナ接続失敗時は、入力フィールドで手動入力に切り替わる（FR-EV-008 テキスト入力）。

## 2. 計測器 IF（IF-006）

### 2-1. 接続方式

| 接続種別 | 対応機器例 | プロトコル |
|---|---|---|
| Bluetooth GATT | BLE 対応ノギス・温度計 | BLE GATT（カスタムサービス）|
| USB シリアル（CDC-ACM）| RS-232 → USB 変換アダプタ | Web USB API または RN Bridge |
| 手動入力 | 全計測器（フォールバック）| — |

### 2-2. 測定値の UCUM 単位変換

```typescript
// 計測器から受信した値の単位変換
function convertMeasurement(
  rawValue: number,
  deviceUnit: string,  // 計測器の単位
  displayUnit: string, // 表示単位（UI 設定）
  siUnit: string,      // SI 基本単位（保存用）
): Measurement {
  const inSi = convertToSi(rawValue, deviceUnit);
  const forDisplay = convertFromSi(inSi, displayUnit);
  
  return {
    measured_value: inSi,    // SI 基本単位で保存（BR-BUS-030）
    unit_ucum: siUnit,
    display_value: forDisplay,
    display_unit_ucum: displayUnit,
  };
}
```

---

**本節で確定した方針**
- **スキャナ（IF-005）は Bluetooth HID 優先・USB HID 代替・ソフトウェアデコードフォールバックの 3 段接続方式とし、GS1 AI 解釈でロット ID・製品コードを取得する。**
- **計測器（IF-006）は BLE GATT / USB シリアル / 手動入力の 3 方式とし、測定値は UCUM SI 基本単位で保存・表示単位は換算後で表示する設計を確定した。**
