# 13 実装方針カード集（FR × 86 件分の方針カード）

本章の責務は、全 86 機能要件（FR-NV/EV/ST/MA/SY/KZ/AU/UI）に対して「実装方針カード」を 1 枚ずつ発行し、設計 ID（API/TBL/MOD/SEQ/SCR/ERR/CFG）との対応を確定することである。本章が DTM M1/M2 の主要入力源となる。

**本節で確定した方針**
- **全 86 FR に対して実装方針カードを発行し、主 API・主 TBL・主 MOD・SEQ・SCR・ERR・CFG の 7 観点での設計対応を確定した。**
- **FR-KZ-008（個人別ランキング）は [Won't: BR-BUS-029] として明示し、概要設計の全 86 FR を取りこぼしなくカバーしたことを確認した。**
- **FR-UI 系 11 件はサーバーサイド API に依存しない「クライアント処理」として設計し、オフライン時も完全に動作することを担保した。**
