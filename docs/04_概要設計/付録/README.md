# 04_概要設計 付録

概要設計書の横断的な識別子管理・トレーサビリティ・変更管理・IPA カバレッジ監査を目的とした補助文書群。

---

## 構成

| ファイル | 目的 |
|---|---|
| `00_設計IDルールと採番規約.md` | 設計識別子 16 種のプレフィックス・採番形式・廃止規約の権威定義（`03_識別子体系と採番規約.md` の詳細版） |
| `01_設計トレーサビリティマトリクス（DTM）.md` | 要件 ID → 設計 ID の 6 マトリクス（M1〜M6）。概要設計の完全性を検証するための中枢文書 |
| `02_RTM拡張版_要件_設計識別子マッピング.md` | 既存要件 RTM（8 列）に「主 SCR・主 API・主 TBL」の 3 列を追加した拡張ビュー。要件定義/付録/01_RTM の設計部補完 |
| `03_IPA共通フレーム概要設計プロセスカバレッジマトリクス.md` | IPA 共通フレーム 2013 の 2.3/2.4 全タスクと本書 10 サブの対応・カバレッジ率 |
| `04_上流文書との対応一覧.md` | 構想 14 章・計画 20 章・要件定義 10 サブ → 04 概要設計 各サブへの引き継ぎ表 |
| `05_設計判断ADR索引.md` | 概要設計フェーズで起票する ADR の索引（本体は `docs/01_管理/ADR/` に格納） |
| `06_変更管理と版数規約.md` | 概要設計書の SemVer 管理・変更種別・Frozen ライフサイクル |
| `99_設計識別子採番台帳.md` | TBL/IDX/VW/API/MOD/CMP/TRN/BAT/MSG/ERR/LOG/KEY/CFG/SEQ/MET/PRM の全件採番台帳 |

---

## 閲覧順

1. `00_設計IDルールと採番規約.md`（識別子を理解する前提）
2. `99_設計識別子採番台帳.md`（現在の採番状況を把握する）
3. `01_設計トレーサビリティマトリクス（DTM）.md`（要件 → 設計の完全性を確認する）
4. `02_RTM拡張版.md`（要件 RTM との整合を確認する）
5. `03_IPA カバレッジマトリクス.md`（IPA プロセス準拠状況を確認する）
6. `04_上流文書との対応一覧.md`（引き継ぎ確認）
7. `05_ADR 索引.md`（設計判断の根拠を確認する）
8. `06_変更管理と版数規約.md`（変更時の手順を確認する）
