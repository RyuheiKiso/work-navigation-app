# 01_システム方式設計

IPA 共通フレーム 2013「2.3 システム方式設計プロセス」の全タスクをカバーする設計成果物の格納場所。論理サブシステム分割・HW/MW スタック選定・配置設計・ネットワーク構成・可用性設計を確定し、後続のソフトウェア方式設計（02 サブ）および詳細設計（05 ディレクトリ）の骨格を与える。
