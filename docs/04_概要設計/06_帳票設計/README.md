# 06_帳票設計

IPA 共通フレーム 2013「2.4 出力設計タスク」に対応する。

要件定義フェーズで確定した 6 帳票（RP-001〜006）のレイアウト・生成パイプライン・改ざん防止・保管を確定する。

---

## 担当する上流要件

| 種別 | 範囲 |
|---|---|
| 帳票 RP | RP-001〜006 全 6 件 |
| FR-AU | FR-AU-004/005（監査ログ/XES エクスポート）|
| NFR-OPS | NFR-OPS-035（7 年保存）|
| NFR-ETH | 個人別集計禁止（RP-004/006）|
