# 13 RP-009 仕入先品質実績レポート設計

本章は帳票 RP-009「仕入先品質実績レポート」のレイアウト・データフィールド・生成規則を確定する。RP-009 は仕入先別・材料別・月次の検査結果を集計し、不合格率・検査厳しさ推移・傾向を品質担当へ提供する定期レポートである。

---

## 1. 帳票概要

| 項目 | 内容 |
|---|---|
| 帳票 ID | RP-009 |
| 帳票名 | 仕入先品質実績レポート |
| 準拠規格 | ISO 9001:2015 §8.4.1（外部提供者の管理・評価） |
| 主目的 | 仕入先別品質実績の月次モニタリング・品質評価（個人別集計禁止の BR-BUS-038 適用） |
| 生成単位 | 仕入先 × 材料 × 月次 1 レポート（BAT-012 月次バッチ生成） |
| 出力形式 | PDF/A-3 + CSV エクスポート |
| 保存期間 | 5 年 |
| 担当要件 | FR-IQ-014, FR-IQ-015, BR-BUS-038, NFR-ETH-002 |

---

## 2. レイアウト構造

### 2-1. サマリー部

| フィールド名 | 表示ラベル |
|---|---|
| supplier_name | 仕入先名 |
| material_name | 材料名 |
| report_month | 集計月 |
| total_lots | 検査ロット数 |
| passed_lots | 合格ロット数 |
| defect_rate_pct | 不合格率（%） |
| current_severity | 現在の検査厳しさ |

### 2-2. 月次推移グラフ（PDF 内グラフ）

不合格率 12 ヶ月推移・検査厳しさ状態変化を折れ線グラフで出力。個人別集計は含まない（NFR-ETH-002 遵守）。

### 2-3. 判定区分内訳

| フィールド名 | 表示ラベル |
|---|---|
| passed_count | PASSED 件数 |
| conditional_pass_count | 特採件数 |
| screening_count | 選別件数 |
| rejected_count | 返品件数 |
