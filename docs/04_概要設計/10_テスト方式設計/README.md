# 10 テスト方式設計

本サブ（10_テスト方式設計）は IPA 共通フレーム 2013「2.3 システム方式設計プロセス」および「2.4 ソフトウェア方式設計プロセス」の評価タスクに対応する。上流の NFR-QUA（品質保証非機能要件）・テスト・受入要件（TST-020〜022 他）・FR 全 86 件・BR-BUS 全 45 件・UC 全 22 件・ALCOA+ 9 原則を受けて、テストレベル・自動化基盤・品質ゲート・ALCOA+ 検証・非機能テスト・受入テストの設計命題を概要設計として確定する。
