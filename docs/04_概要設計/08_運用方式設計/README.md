# 08 運用方式設計

本サブ（08_運用方式設計）は IPA 共通フレーム 2013「2.3 システム方式設計プロセス」の運用方式設計タスクに対応する。上流の NFR-OPS 全件・NFR-AVL-001〜020 を受けて、IT 担当 1 名の兼務体制で本システムを安定稼働させるための監視・ロギング・バックアップ・ジョブ管理・縮退・アラートの設計命題を確定する。本サブは「個人開発・個人運用」を不変の前提とし、過剰な監視基盤の導入を禁止する。
