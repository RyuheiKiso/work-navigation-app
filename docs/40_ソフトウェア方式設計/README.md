# 40_ソフトウェア方式設計

IPA 共通フレーム 2013 (SLCP-JCF2013) における **ソフトウェア方式設計プロセス** に該当する設計成果物を集約する。

「何を作るか」(要件定義) と「どう実装するか」(詳細設計) の間に位置し、本システムを構成するソフトウェアコンポーネントの **構造・責務分離・依存方向・公開境界** を決定することを目的とする。
