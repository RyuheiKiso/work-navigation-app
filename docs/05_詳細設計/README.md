## 本書の前提

- **コード粒度**: SQL DDL 全文・Rust struct/trait/fn シグネチャ・TypeScript interface を各章のコードブロックに記載する。実装ファイル（`src/`）の作成は対象外。
- **上流固定**: `04_概要設計` が確定した 31 モジュール（MOD-NNN）・35 テーブル（TBL-NNN）・39 API エンドポイント（API-NNN）・86 機能要件（FR-NNN）は変更しない。
- **識別子継承**: TBL-001〜035 は `04_概要設計/付録/99_設計識別子採番台帳.md` を継承。本書では IDX/VW/TST/FNC を新規採番する（`01_詳細設計の識別子規約と採番台帳.md` で管理）。
- **図**: drawio-authoring スキルで全図作成。drawio-lint / svg-postcheck ERROR ゼロを完成条件とする。
