# 09 エラーハンドリング詳細設計

本章は ERR-NNN 識別子全 22 件の完全カタログ・AppError enum・RFC 9457 Problem Details への IntoResponse 実装・ログ出力設計を確定する。全エラーは `09_エラーハンドリング統一方式.md` の上流設計を継承し、コーディングに直接利用できる型定義として精緻化する。

## 1. AppError enum（全 22 件）

```rust
// crates/wnav_api/src/error.rs

use thiserror::Error;
use uuid::Uuid;

/// アプリケーション全体のエラー型。
/// `thiserror` で実装し、`IntoResponse` で RFC 9457 Problem Details に変換する。
#[derive(Debug, Error)]
pub enum AppError {
    // --- ERR-AUTH: 認証認可エラー ---

    /// ERR-AUTH-001: JWT が存在しない、または形式不正
    #[error("ERR-AUTH-001: Authentication required")]
    Unauthorized,

    /// ERR-AUTH-002: 操作権限なし（一般的な 403）
    #[error("ERR-AUTH-002: Access denied")]
    Forbidden,

    /// ERR-AUTH-003: JWT の exp が過去
    #[error("ERR-AUTH-003: JWT expired")]
    JwtExpired,

    /// ERR-AUTH-004: RBAC 判定失敗（BR-BUS-040/042）
    #[error("ERR-AUTH-004: Insufficient role")]
    InsufficientRole {
        required_roles: Vec<String>,
        current_roles: Vec<String>,
    },

    // --- ERR-VAL: バリデーションエラー ---

    /// ERR-VAL-001: 必須フィールド欠損（BR-BUS-031）
    #[error("ERR-VAL-001: Required field missing: {field}")]
    RequiredFieldMissing { field: String },

    /// ERR-VAL-002: 数値範囲外（BR-BUS-030）
    #[error("ERR-VAL-002: Numeric out of range: {field} = {value}")]
    NumericOutOfRange { field: String, value: String },

    /// ERR-VAL-003: 単位不一致またはフォーマットエラー（BR-BUS-033/034/035）
    #[error("ERR-VAL-003: Unit mismatch or format error: {field}")]
    UnitMismatchOrFormatError { field: String },

    /// ERR-VAL-004: テキスト最大長超過（BR-BUS-032）
    #[error("ERR-VAL-004: Text max length exceeded: {field} (max {max_length})")]
    TextMaxLengthExceeded { field: String, max_length: usize },

    // --- ERR-BIZ: 業務ルール違反 ---

    /// ERR-BIZ-001: ロックステップ違反（BR-BUS-001）
    #[error("ERR-BIZ-001: Step sequence violation (current={current_step}, attempted={attempted_step})")]
    StepSequenceViolation {
        current_step: u32,
        attempted_step: u32,
    },

    /// ERR-BIZ-002: 証拠記録必須（BR-BUS-003）
    #[error("ERR-BIZ-002: Evidence is required for this step")]
    EvidenceRequired,

    /// ERR-BIZ-003: 電子サイン必須（BR-BUS-004）
    #[error("ERR-BIZ-003: Electronic signature is required")]
    SignRequired,

    /// ERR-BIZ-004: 校正期限切れ（BR-BUS-007）
    #[error("ERR-BIZ-004: Calibration expired for device: {device_id}")]
    CalibrationExpired { device_id: Uuid },

    /// ERR-BIZ-005: SOP バージョンがフリーズ（BR-BUS-008）
    #[error("ERR-BIZ-005: SOP version is frozen (published/archived): {version_id}")]
    SopVersionFrozen { version_id: Uuid },

    /// ERR-BIZ-006: スキルレベル不足（BR-BUS-002/041）
    #[error("ERR-BIZ-006: Insufficient skill level (required={required}, current={current})")]
    SkillLevelInsufficient { required: u8, current: u8 },

    // --- ERR-DB: データベースエラー ---

    /// ERR-DB-001: Idempotency Key の重複（冪等応答）
    #[error("ERR-DB-001: Idempotency replay conflict for key: {key}")]
    IdempotencyReplayConflict { key: Uuid },

    /// ERR-DB-002: 楽観ロック競合
    #[error("ERR-DB-002: Optimistic lock conflict (resource was updated concurrently)")]
    OptimisticLockConflict,

    /// ERR-DB-003: ハッシュチェーン破損（致命的）
    #[error("ERR-DB-003: Hash chain broken at block: {block_id:?}")]
    HashChainBroken { block_id: Option<Uuid> },

    // --- ERR-EXT: 外部連携エラー ---

    /// ERR-EXT-001: マスタ同期不可
    #[error("ERR-EXT-001: Master sync unavailable (upstream: {upstream})")]
    MasterSyncUnavailable { upstream: String },

    /// ERR-EXT-002: Webhook 配信失敗
    #[error("ERR-EXT-002: Webhook delivery failed (endpoint: {endpoint})")]
    WebhookDeliveryFailed { endpoint: String },

    // --- ERR-SYS: システムエラー ---

    /// ERR-SYS-001: 予期しないサーバーエラー
    #[error("ERR-SYS-001: Internal server error")]
    InternalServerError(#[source] anyhow::Error),

    /// ERR-SYS-002: レート制限超過（CFG-002）
    #[error("ERR-SYS-002: Rate limit exceeded")]
    RateLimited,

    /// リソースが見つからない（404。ERR-NNN 未採番の汎用 Not Found）
    #[error("Resource not found")]
    NotFound,

    /// Idempotency-Key ヘッダが必須の書き込みリクエストで欠如している
    #[error("Idempotency-Key header is required for write requests")]
    MissingIdempotencyKey,
}
```

## 3. IntoResponse 実装（AppError → RFC 9457）

```rust
// crates/wnav_api/src/error.rs（続き）

use axum::{
    http::StatusCode,
    response::{IntoResponse, Response},
    Json,
};

impl AppError {
    /// ERR-NNN 識別子を返す。
    pub fn error_code(&self) -> &'static str {
        match self {
            Self::Unauthorized              => "ERR-AUTH-001",
            Self::Forbidden                 => "ERR-AUTH-002",
            Self::JwtExpired                => "ERR-AUTH-003",
            Self::InsufficientRole { .. }   => "ERR-AUTH-004",
            Self::RequiredFieldMissing { .. }    => "ERR-VAL-001",
            Self::NumericOutOfRange { .. }       => "ERR-VAL-002",
            Self::UnitMismatchOrFormatError { .. }=> "ERR-VAL-003",
            Self::TextMaxLengthExceeded { .. }   => "ERR-VAL-004",
            Self::StepSequenceViolation { .. }   => "ERR-BIZ-001",
            Self::EvidenceRequired               => "ERR-BIZ-002",
            Self::SignRequired                   => "ERR-BIZ-003",
            Self::CalibrationExpired { .. }      => "ERR-BIZ-004",
            Self::SopVersionFrozen { .. }        => "ERR-BIZ-005",
            Self::SkillLevelInsufficient { .. }  => "ERR-BIZ-006",
            Self::IdempotencyReplayConflict { .. } => "ERR-DB-001",
            Self::OptimisticLockConflict         => "ERR-DB-002",
            Self::HashChainBroken { .. }         => "ERR-DB-003",
            Self::MasterSyncUnavailable { .. }   => "ERR-EXT-001",
            Self::WebhookDeliveryFailed { .. }   => "ERR-EXT-002",
            Self::InternalServerError(_)         => "ERR-SYS-001",
            Self::RateLimited                    => "ERR-SYS-002",
            Self::NotFound                       => "ERR-NOT-FOUND",
            Self::MissingIdempotencyKey          => "ERR-VAL-IDEMPOTENCY",
        }
    }

    /// HTTP ステータスコードを返す。
    pub fn status_code(&self) -> StatusCode {
        match self {
            Self::Unauthorized | Self::JwtExpired       => StatusCode::UNAUTHORIZED,
            Self::Forbidden | Self::InsufficientRole {..} | Self::SkillLevelInsufficient {..}
                                                         => StatusCode::FORBIDDEN,
            Self::RequiredFieldMissing {..}
            | Self::NumericOutOfRange {..}
            | Self::UnitMismatchOrFormatError {..}
            | Self::TextMaxLengthExceeded {..}
            | Self::EvidenceRequired
            | Self::SignRequired
            | Self::CalibrationExpired {..}              => StatusCode::UNPROCESSABLE_ENTITY,
            Self::StepSequenceViolation {..}
            | Self::SopVersionFrozen {..}
            | Self::IdempotencyReplayConflict {..}
            | Self::OptimisticLockConflict               => StatusCode::CONFLICT,
            Self::RateLimited                            => StatusCode::TOO_MANY_REQUESTS,
            Self::NotFound                               => StatusCode::NOT_FOUND,
            Self::MissingIdempotencyKey                  => StatusCode::UNPROCESSABLE_ENTITY,
            Self::MasterSyncUnavailable {..}             => StatusCode::SERVICE_UNAVAILABLE,
            Self::WebhookDeliveryFailed {..}             => StatusCode::BAD_GATEWAY,
            Self::HashChainBroken {..}
            | Self::InternalServerError(_)               => StatusCode::INTERNAL_SERVER_ERROR,
        }
    }

    /// tracing ログレベルを返す。
    pub fn log_level(&self) -> LogLevel {
        match self {
            Self::HashChainBroken {..}
            | Self::InternalServerError(_) => LogLevel::Error,
            Self::Forbidden
            | Self::InsufficientRole {..}
            | Self::OptimisticLockConflict
            | Self::MasterSyncUnavailable {..}
            | Self::WebhookDeliveryFailed {..}
            | Self::CalibrationExpired {..}
            | Self::SkillLevelInsufficient {..} => LogLevel::Warn,
            _ => LogLevel::Info,
        }
    }

    /// LOG-NNN 識別子を返す。
    pub fn log_id(&self) -> &'static str {
        match self.log_level() {
            LogLevel::Error => "LOG-ERR-001",
            LogLevel::Warn  => "LOG-WARN-001",
            LogLevel::Info  => "LOG-INFO-001",
        }
    }

    /// i18n キーを使用したユーザー向けメッセージを返す。
    /// 呼び出し元のロケール解決が完了している前提。
    pub fn user_message(&self) -> String {
        let key = match self {
            Self::Unauthorized                  => "error.auth.unauthorized",
            Self::Forbidden                     => "error.auth.forbidden",
            Self::JwtExpired                    => "error.auth.jwt_expired",
            Self::InsufficientRole {..}         => "error.auth.insufficient_role",
            Self::RequiredFieldMissing {..}     => "error.val.required_field_missing",
            Self::NumericOutOfRange {..}        => "error.val.numeric_out_of_range",
            Self::UnitMismatchOrFormatError {..}=> "error.val.unit_mismatch",
            Self::TextMaxLengthExceeded {..}    => "error.val.text_max_length",
            Self::StepSequenceViolation {..}    => "error.biz.lock_step_violation",
            Self::EvidenceRequired              => "error.biz.evidence_required",
            Self::SignRequired                  => "error.biz.sign_required",
            Self::CalibrationExpired {..}       => "error.biz.calibration_expired",
            Self::SopVersionFrozen {..}         => "error.biz.sop_version_frozen",
            Self::SkillLevelInsufficient {..}   => "error.biz.skill_level_insufficient",
            Self::IdempotencyReplayConflict {..}=> "error.db.idempotency_conflict",
            Self::OptimisticLockConflict        => "error.db.optimistic_lock",
            Self::HashChainBroken {..}          => "error.db.hash_chain_broken",
            Self::MasterSyncUnavailable {..}    => "error.ext.master_sync_unavailable",
            Self::WebhookDeliveryFailed {..}    => "error.ext.webhook_failed",
            Self::InternalServerError(_)        => "error.sys.internal",
            Self::RateLimited                   => "error.sys.rate_limited",
            Self::NotFound                      => "error.general.not_found",
            Self::MissingIdempotencyKey         => "error.general.missing_idempotency_key",
        };
        // デフォルトロケール "ja" でメッセージを取得
        crate::locale::t(key, "ja")
    }
}

/// ログレベル（tracing への橋渡し）
pub enum LogLevel {
    Error,
    Warn,
    Info,
}

impl IntoResponse for AppError {
    fn into_response(self) -> Response {
        let request_id = Uuid::now_v7();
        let status = self.status_code();

        // エラーレベルに応じたログ出力
        match self.log_level() {
            LogLevel::Error => tracing::error!(
                log_id = self.log_id(),
                error_code = self.error_code(),
                detail = %self,
            ),
            LogLevel::Warn => tracing::warn!(
                log_id = self.log_id(),
                error_code = self.error_code(),
                detail = %self,
            ),
            LogLevel::Info => tracing::info!(
                log_id = self.log_id(),
                error_code = self.error_code(),
            ),
        }

        let body = ProblemDetails {
            type_: format!(
                "https://errors.wnav.example.com/{}",
                self.error_code()
            ),
            title: self.error_code().to_string(),
            status: status.as_u16(),
            detail: self.user_message(),
            instance: format!("/requests/{}", request_id),
            error_id: Some(self.error_code().to_string()),
        };

        (
            status,
            [("Content-Type", "application/problem+json")],
            Json(body),
        )
            .into_response()
    }
}
```

## 4. エラー変換実装

```rust
// crates/wnav_api/src/error.rs（続き）

/// sqlx::Error から AppError への変換
impl From<sqlx::Error> for AppError {
    fn from(e: sqlx::Error) -> Self {
        match e {
            sqlx::Error::RowNotFound => AppError::NotFound,
            _ => AppError::InternalServerError(anyhow::anyhow!(e)),
        }
    }
}

/// wnav_domain::DomainError から AppError への変換
impl From<wnav_domain::error::DomainError> for AppError {
    fn from(e: wnav_domain::error::DomainError) -> Self {
        match e.code.as_str() {
            "ERR-BIZ-001" => AppError::StepSequenceViolation {
                current_step: e.context.get("current_step")
                    .and_then(|v| v.as_u64()).unwrap_or(0) as u32,
                attempted_step: e.context.get("attempted_step")
                    .and_then(|v| v.as_u64()).unwrap_or(0) as u32,
            },
            "ERR-BIZ-002" => AppError::EvidenceRequired,
            "ERR-BIZ-003" => AppError::SignRequired,
            "ERR-DB-001"  => AppError::IdempotencyReplayConflict {
                key: e.context.get("key")
                    .and_then(|v| v.as_str())
                    .and_then(|s| Uuid::parse_str(s).ok())
                    .unwrap_or(Uuid::nil()),
            },
            "ERR-DB-002"  => AppError::OptimisticLockConflict,
            _             => AppError::InternalServerError(
                anyhow::anyhow!("DomainError: {:?}", e)
            ),
        }
    }
}
```

**本節で確定した方針**
- **ERR-AUTH（4 件）・ERR-VAL（4 件）・ERR-BIZ（6 件）・ERR-DB（3 件）・ERR-EXT（2 件）・ERR-SYS（2 件）の計 22 件の ERR-NNN カタログを確定し、HTTP ステータス・ログレベル・i18n キー・retry 可否を一元管理した。**
- **AppError::IntoResponse は RFC 9457 Problem Details に変換し、`Content-Type: application/problem+json` で返すことを確定した。**
- **sqlx::Error と DomainError から AppError への変換（From trait）を実装し、Infrastructure 層エラーが自動的に適切な ERR-NNN に変換される設計を確定した。**
