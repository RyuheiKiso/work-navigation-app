# 01 ハンディ APP 画面項目定義（15 画面）

本章は SCR-HA-001〜015 の全 15 画面について、画面項目定義（項目 ID・型・必須・制約・バリデーション・対応 FR）を確定する。React Native 実装のコーディング直前仕様として機能する。

---

## 1. 項目型定義

| 型 | 説明 |
|---|---|
| TEXT | 文字列入力フィールド |
| PASSWORD | マスク表示文字列入力 |
| NUMERIC | 数値入力フィールド |
| BUTTON | タップアクション |
| LIST | スクロール可能なリスト |
| CAMERA | カメラプレビュービュー |
| SIGN_PAD | 手書きサインパッド |

| BADGE | ステータスバッジ表示 |

| SELECT | ドロップダウン / ピッカー |

**本節で確定した方針**
- **SCR-HA-001〜015 の全 15 画面について、全項目（F{SCR}-NNN）・型・必須・制約・バリデーション・対応 FR・遷移 TRN を確定した。**
- **全 15 画面に CMP-HA-006（OfflineBanner）を配置し、オフライン時の縮退表示（FR-SY-008）を全画面で担保することを確定した。**
- **SCR-HA-005/006/007 には CMP-HA-008（AndonButton）を常設配置し、作業中どの Step からでも FR-KZ-001（アンドン発報）を起動できることを確定した。**
