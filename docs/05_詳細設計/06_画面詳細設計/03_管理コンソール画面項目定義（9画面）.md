# 03 管理コンソール画面項目定義（9 画面）

本章は SCR-MC-001〜009 の全 9 画面について、画面項目定義（項目 ID・型・必須・制約・バリデーション・対応 FR/NFR）を確定する。React 実装のコーディング直前仕様として機能する。

---

## 1. 管理コンソール固有の項目型

| 型 | 説明 |
|---|---|
| GAUGE | SLI 値ゲージ（CMP-MC-001 ベース）|
| CHART | 時系列チャート表示 |
| TABLE | ページネーション付きデータグリッド |
| FILTER_FORM | 複合フィルタフォーム |
| SIGN_PAD | 手書きサインパッド |
| DATE_RANGE | 日付範囲ピッカー |
| PROGRESS | 進捗インジケーター |

**本節で確定した方針**
- **SCR-MC-001〜009 の全 9 画面について、全項目（F2{SCR番号2桁}-NNN）・型・必須・制約・バリデーション・対応 FR/NFR・遷移 TRN を確定した。**
- **SCR-MC-001（運用ダッシュボード）は自動更新間隔（10s / 30s / 60s / 5m）を SELECT で設定可能とし、OPS-041 を担保することを確定した。**
- **SCR-MC-008（ハッシュチェーン検証）は CMP-MC-003（HashChainVerifyResult）を中心に破断ブロックの期待ハッシュ・実ハッシュを全件表示し、FR-AU-006 の完全な証跡提供を確定した。**
