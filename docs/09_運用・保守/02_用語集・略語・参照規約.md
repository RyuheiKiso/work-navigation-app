# 02 用語集・略語・参照規約

本章は `09_運用・保守` フェーズで使用する運用固有用語・略語・文書間参照規約を確定する。本章に定義した用語は 09 章全体で統一的に使用し、定義外の略語・略称は使用しない。

---

## 5. 参照規約

### 5-1. 文書間参照の形式

```markdown
# 09 章内文書間の参照
[03_運用体制と役割実装.md §3](../03_運用体制と役割実装.md)

# 上流文書への参照（読み取りのみ）
[../04_概要設計/08_運用方式設計/01_運用概要設計.md](../04_概要設計/08_運用方式設計/01_運用概要設計.md)

# ポストモーテム記録への参照
[../../ポストモーテム/2026-MM-DD_P1_<slug>.md](../../ポストモーテム/)

# 業界分析への参照
[../../90_業界分析/13_安全文化と安全管理システム.md](../../90_業界分析/13_安全文化と安全管理システム.md)
```

### 5-2. 識別子参照の形式

```markdown
# 要件参照
NFR-OPS-001（稼働率 ≥ 99.5%）

# 手順参照
OPS-PROC-002（週次定期作業手順）

# 障害記録参照
INC-2026-001 / PM-2026-001

# 保守記録参照
REC-001

# 変更要求参照
CR-OPS-001

# 運用 ADR 参照
ADR-OPS-001
```

**本節で確定した方針**
- **本章の用語集に定義した 50 語以上を 09 章全体で統一使用する。定義外の略称は使用しない。**
- **文書間参照は相対パス形式とし、絶対パスを使用しない。**
- **識別子参照は「プレフィックス-番号（概要）」形式を推奨する。**
