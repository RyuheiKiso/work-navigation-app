# 99 運用保守識別子採番台帳

本台帳は `09_運用・保守` フェーズで採番した全識別子の状態を一元管理する権威台帳である。CHK/RUN/INC/PM/REC/CR-OPS/ADR-OPS の 7 種プレフィックスを対象とし、欠番禁止を厳守する。

各識別子の追加・状態変更・廃止時にはこの台帳を更新してからドキュメントを変更する（ドキュメント先行原則）。

---

## 凡例

| フィールド | 内容 |
|---|---|
| ID | 識別子（例: CHK-001）|
| 分類 / 概要 | 識別子が表す項目の分類・概要 |
| 対応 OPS-PROC | 関連する運用手順識別子 |
| 状態 | 予約済み（未実施）/ 確定済み / 廃止 |
| 次採番値 | 次回採番する番号（各セクション先頭に記載）|
