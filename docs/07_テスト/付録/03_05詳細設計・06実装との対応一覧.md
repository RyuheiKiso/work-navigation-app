## 3. 参照方向の原則

本書（07_テスト）からの参照は**読み取り専用**である。上流フェーズ（05_詳細設計・06_実装）のファイルは本書から変更しない。

```
参照方向（許可）:
  07_テスト → 05_詳細設計（テストケース定義の参照）
  07_テスト → 06_実装（ビルド・環境・方針の参照）
  07_テスト → 04_概要設計（テスト方式・品質ゲートの参照）
  07_テスト → 03_要件定義（受入基準・TENV 要件の参照）

参照方向（禁止）:
  07_テスト から上流文書への変更・上書き
  07_テスト 内での上流識別子（FR/TST/TENV）の再定義
```

**本節で確定した方針**
- **05_詳細設計/08_ の 8 ファイルが確定したテストケース（116 件）を 07_テスト の各実施手順書が引き継ぎ、実施コマンド・実施順序・結果記録フォーマットを追加することを確定した。**
- **06_実装との対応は「読み取り専用参照」に限定し、07_テスト から上流文書を変更することを禁止することを確定した。**
- **上流固定値（86 FR / 116 TST / 8 TENV / 22 UC / 45 BR-BUS）は本書で一切変更しないことを確定した。**
