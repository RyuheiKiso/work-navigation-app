# 付録

本付録は `07_テスト/` 全体で共有する識別子規約・トレーサビリティマトリクス・IPA カバレッジ・変更管理規約・ADR 索引・採番台帳を一元管理する。

---

## §1 本付録の責務

本付録は以下の 3 点を担う。

1. **テスト識別子体系の権威定義**: 07_テスト で新規定義する識別子（TST-PLAN / TST-EXEC / DEFECT / RGATE / ADR-TEST / CR-TEST）の採番規約を確定し、上流継承識別子（TST-NNN / TENV-NNN）との体系整合を保証する。
2. **トレーサビリティの完全性保証**: 86 FR × 116 TST-NNN × 実行記録（TST-EXEC）× 欠陥（DEFECT）の 7 段トレーサビリティマトリクス（TETM）を管理し、全 FR のテストカバレッジが 100% であることを本付録で検証可能にする。
3. **IPA 共通フレーム準拠の証明**: SLCP-JCF2013 テストプロセス群の全タスクと 07_テスト 各章との対応を明示し、フレーム準拠の証拠文書とする。
