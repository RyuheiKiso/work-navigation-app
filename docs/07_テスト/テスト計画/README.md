# テスト計画

本サブ（テスト計画）は IPA 共通フレーム 2013「6.4.1 テスト計画プロセス」に対応する成果物群を管理する。
上流の要件定義（03_要件定義/テスト・受入要件）・テスト方式設計（04_概要設計/10_テスト方式設計）・テストケース詳細設計（05_詳細設計/08_テストケース詳細設計）を受けて、実行計画・環境・データ・判定基準・スケジュールを一元的に確定する。
