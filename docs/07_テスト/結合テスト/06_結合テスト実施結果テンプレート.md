# 06 結合テスト実施結果テンプレート

本章は SLCP-JCF2013 §6.1.1.3 および §6.1.2.4 に対応する L2 統合テスト・L4 契約テストの実施結果を記録するためのテンプレートを提供する。
テスト実施者は本テンプレートをコピーして `TST-EXEC-YYYYMMDD-NNN` の識別子を付与した上で記録する。

---

## テンプレートの使い方

1. 本ファイルをコピーして `TST-EXEC-YYYYMMDD-NNN.md` と命名する（例: `TST-EXEC-20260518-001.md`）
2. 全フィールドを埋める。空欄禁止とする
3. 実施後に本サブの `results/` ディレクトリに保存する

**本節で確定した方針**
- **本テンプレートの全フィールドを記入することを結合テスト完了の必要条件とする。空欄が存在する場合は完了と認めない。**
- **総合判定のいずれか 1 項目が FAIL の場合は総合判定を FAIL とする。全項目 PASS の場合のみ総合判定 PASS とする。**
- **実施記録は `results/TST-EXEC-YYYYMMDD-NNN.md` として保存し、関連する DEFECT-NNN との相互参照リンクを記載することを求める。**
