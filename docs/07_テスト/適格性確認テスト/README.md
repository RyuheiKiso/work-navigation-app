# 適格性確認テスト

本サブ（適格性確認テスト）は IPA 共通フレーム 2013（SLCP-JCF2013）の以下 3 タスクに対応する成果物群を管理する。

| SLCP-JCF2013 タスク | 対応範囲 |
|---|---|
| §6.1.1.4 ソフトウェア適格性確認テスト | E2E テスト 22 件（UC 22 件に 1:1 対応）|
| §6.1.2.5 システム適格性確認テスト | 性能・セキュリティ・可用性 NFR テスト |
| §6.4.8 受入支援 | UAT シナリオ A〜E・ブラインドテスト・サインオフ |

---

## 本サブの責務

本サブは以下を確定する。

- **ソフトウェア適格性確認**: TENV-003（Playwright）・TENV-004（Detox）を用いた UC 22 件の E2E テスト実施手順
- **システム適格性確認**: k6 性能テスト（TST-perf-001〜005）・OWASP ZAP セキュリティスキャン（TST-sec-001〜010）・可用性テストの実施手順
- **ALCOA+ 検証**: 21 CFR Part 11、PIC/S PE 009-16、ISO 19005-3、IEEE XES 2.0 準拠の 9 属性検証手順（TST-alcoa-001〜009）
- **受入相当テスト**: 個人開発での自己 UAT シナリオ A〜E の実施手順・合否判定・電子サインオフ手順
- **リリース判定**: RGATE-001〜006 全 PASS → UAT 合格 → FR-AU-001 準拠サインオフ → ver1.0.0 リリース
