## 7. 公開コミュニケーション

- GitHub Discussions で本報告へのリンクを共有する。
- CHANGELOG にも金額の概略（合計のみ）を記録する。
- 詳細は本書で公開する。

## 8. 受入観点（§19.5.4）

- 各資金チャネルの採否が `MAINTAINERS.md` で公開されていること。
- 四半期収支報告が本テンプレートに従い `docs/governance/financial-report-*.md` に存在すること。
- メンテナ可用時間予算が公開され、PR レビュー SLA（§19.4）と整合していること。
- 寄贈先候補が §19.4 ＋ §22.3 撤退条件と双方向参照されていること。
- 単独大口寄付発生時に §29 リスク登録簿が更新されていること（R-017）。
