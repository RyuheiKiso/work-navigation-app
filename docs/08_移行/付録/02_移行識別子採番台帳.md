# 02 移行識別子採番台帳

本章の責務は、`docs/08_移行/` で新規採番する全識別子（MIG-X / MIG-RH / MIG-RB / MIG-CK / MIG-REP / D-MIG-X / CON-MIG-X）の状態を権威的に管理することである。本台帳と本文の不整合が発生した場合は本台帳を正とする。欠番・重複は禁止する。廃止は行削除でなく `[廃止: YYYY-MM-DD 廃止理由: ...]` を行頭付与で管理する。
