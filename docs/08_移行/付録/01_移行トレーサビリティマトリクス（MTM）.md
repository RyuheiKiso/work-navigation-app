# 01 移行トレーサビリティマトリクス（MTM）

本章の責務は、移行要件 MIG-001〜074 から移行方式設計 DES-MIG → 08 章実施命題 MIG-X/RH/RB → 検証 MIG-CK → 報告 MIG-REP へのトレーサビリティを正典的に管理することである。本 MTM が移行要件と実施・検証・報告の間の双方向リンクの唯一の権威ソースである。

---

## §1 MTM の目的と利用方法

### 目的

MTM（Migration Traceability Matrix）は移行要件から実施・検証・報告までの全工程において、すべての要件が担当章・識別子によって充足されることを保証する文書である。

MTM の双方向リンクを以下の通り定義する。

**前方リンク（要件 → 設計 → 実施）**: MIG-NNN（移行要件）→ DES-MIG-NNN（移行方式設計）→ MIG-X-NNN / MIG-RH-NNN / MIG-RB-NNN（08 章実施命題）→ MIG-CK-NNN（チェックリスト）→ MIG-REP-NNN（報告様式）の方向で、要件が実施・検証・報告に反映されていることを確認する。

**後方リンク（実施 → 要件）**: 実施命題 MIG-X-NNN から対応する MIG-NNN（移行要件）を逆引きする方向で、実施命題が要件に根拠を持つことを確認する。後方リンクの索引は §3 に示す DES-MIG → 08 章前方リンク索引で補完する。

### 完全性保証

MIG-001〜074 の全 74 件が MTM の §2 に存在することを完全性の保証条件とする。MTM の §4 に記載する完全性判定手順に従い、定期的に完全性を確認する。

### 利用方法

MTM は以下の場面で参照する。

- **要件レビュー時**: MIG-NNN の「状態」列が全行「確定済み」であることを確認する。
- **設計レビュー時**: DES-MIG 列に担当設計命題 ID が記載されていることを確認する。
- **実施計画承認時**: 08 章実施命題列に MIG-X/RH/RB ID が記載されていることを確認する。
- **検証実施時**: MIG-CK 列の検証項目を実施し結果を記録する。
- **移行完了宣言時**: 全 74 行の「状態」が「確定済み」であることを確認する。

---

**本節で確定した方針**
- 双方向リンク（前方：要件→設計→実施、後方：実施→要件）を MTM の管理対象として確定する。
- MIG-001〜074 全 74 件が MTM に存在することを完全性の保証条件として確定する。
- MTM の権威ソースは本ファイルのみとし、他ファイルの部分的な記載はこれに対応する参照とすることを確定する。

## §4 MTM の完全性判定方法

### 完全性判定の定義

MTM の完全性が保たれているとは、以下の条件が全件成立していることを指す。

1. §2 のマトリクスに MIG-001〜074 の全 74 行が存在すること。
2. 全 74 行の「状態」列が「確定済み」または「予約済み」のいずれかであること。
3. 全 74 行の「DES-MIG ID」列に少なくとも 1 つの DES-MIG 識別子が記載されていること。
4. 全 74 行の「08 章実施命題」列に少なくとも 1 つの MIG-X/RH/RB 識別子または「付録/03 カバレッジ表で対応」の記載があること。
5. 全 74 行の「担当章」列に有効な章ファイルパスが記載されていること。

### 完全性確認の実施タイミング

MTM の完全性確認は以下のタイミングで実施する。

- 移行計画の各章作成完了時
- MIG-NNN または DES-MIG-NNN が追加・廃止された時
- 移行実施フェーズ開始前（Go/No-Go 判定 MIG-063〜065 の前提確認として）
- 移行完了宣言（MIG-060〜062 の条件確認として）

### 完全性確認の手順

以下の手順で MTM の完全性を確認する。

1. §2 の行数を数え、74 行であることを確認する。
2. MIG-001 から MIG-074 まで連番が欠落なく存在することを確認する。
3. 廃止行（`[廃止]` マーカー付き）を除いた有効行数が要件数と一致することを確認する。
4. 「状態」列に「確定済み」または「予約済み」以外の値が存在しないことを確認する。
5. 「DES-MIG ID」「08 章実施命題」「担当章」列に空欄がないことを確認する。

### 完全性の記録

MTM 完全性確認の結果は、移行計画/10（受入レビューと導入結果評価）の確認記録に記載する。Go/No-Go 判定実施前の完全性確認結果は MIG-CK-113〜120 として記録する。

---

**本節で確定した方針**
- MTM 完全性判定の 5 条件をすべて満たすことを完全性の保証条件として確定する。
- 移行実施フェーズ開始前（Go/No-Go 判定の前提）に必ず完全性確認を実施することを確定する。
- 完全性確認の結果を移行計画/10 の確認記録に記載することを確定する。
