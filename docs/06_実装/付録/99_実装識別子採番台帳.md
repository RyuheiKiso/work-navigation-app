# 99 実装識別子採番台帳

本台帳は `06_実装` フェーズで採番した全識別子の状態を一元管理する権威台帳である。IMPL/CR/QA/PROB/ADR-IMPL/ENV の 6 種プレフィックスを対象とし、欠番禁止を厳守する。

各識別子の追加・状態変更・廃止時にはこの台帳を更新してからコードを変更する（ドキュメント先行原則）。

---

## 凡例

| フィールド | 内容 |
|---|---|
| ID | 識別子（例: IMPL-001）|
| 対応 MOD-NNN | 対応するモジュール識別子 |
| 物理名 / 概要 | 対応する実装単位・物理名・変数名等 |
| 状態 | 予約済み（未着手）/ 実装中 / 確定済み / 廃止 |
| 次採番値 | 次回採番する番号（各セクション先頭に記載）|
