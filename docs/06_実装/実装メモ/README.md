## 本ディレクトリの用途

- 外部ドキュメント・RFC・クレートドキュメントへのブックマーク集
- Rust / React Native / axum / TypeORM 等のライブラリチートシート
- 開発環境セットアップメモ（WSL2・Android SDK・iOS シミュレータ等）
- 実装中の一時メモ（確認が取れた時点で権威ドキュメントに昇格させること）
