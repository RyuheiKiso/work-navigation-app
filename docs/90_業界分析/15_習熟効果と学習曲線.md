# 習熟効果と学習曲線

## 1. 学習曲線の発見

T. P. Wright は 1936 年、航空機機体製造のデータを分析し、「**累積生産量が 2 倍になるたびに、直接労働の累積平均単位時間が一定割合 (学習率) 減少する**」という経験的規則性を発見した (*Journal of Aeronautical Sciences*, 1936)。

### 1.1 Wright の学習曲線モデル

```
Y(x) = a · x^b

Y(x) : x 個目を生産したときの累積平均単位時間 (または累積平均コスト)
a     : 初回生産時間 (初回コスト)
x     : 累積生産数
b     : 学習係数 (b = log(r) / log(2); r は学習率)
```

**学習率 (Learning Rate)** は累積生産量が倍増したときのコスト残存率。

| 学習率 | 意味 | 典型的な産業 |
|---|---|---|
| 80% | 倍増ごとにコストが 20% 減少 | 航空機、複雑な機械組立 |
| 85% | 倍増ごとにコストが 15% 減少 | 一般製造業 |
| 90% | 倍増ごとにコストが 10% 減少 | 資本集約型・自動化が進んだ工程 |

### 1.2 単位理論 vs 累積平均理論

| 理論 | 内容 |
|---|---|
| 累積平均理論 (Cumulative Average) | Wright が提唱。x 個の累積平均時間が学習率に従って減少する。 |
| 単位理論 (Unit Theory; Crawford) | x 番目の単位にかかる時間が学習率に従って減少する。 |

両モデルは異なる予測値を出すため、適用する場面 (入札・原価計画・訓練計画) に応じて選択が必要。

## 2. 経験曲線 (Experience Curve) — BCG (1966)

Boston Consulting Group が Wright の学習曲線を拡張し、直接労働費だけでなく**製造・管理・流通を含む全コスト**が累積総生産量に対して冪関数的に低下する「経験曲線 (Experience Curve)」を提唱した。

```
C(Q) = C₀ · Q^(-α)

C(Q) : 累積生産量 Q における単位コスト (実質価格)
C₀   : 初期コスト
α    : 経験指数 (α > 0)
```

### 2.1 戦略的含意

- **市場シェアと累積経験は強く相関する**: 大きな市場シェアを持つ企業は累積経験が多く、競合より低コストを実現できる。
- **BCG プロダクト・ポートフォリオ・マトリックス (PPM)**: 経験曲線に基づく市場成長率とシェアの 2 軸マトリックス (スター / 金の卵 / 問題児 / 負け犬)。
- **先行者利益 (First-Mover Advantage)**: 早期参入による累積経験の蓄積がコスト障壁となる。

### 2.2 批判

- 経験曲線はあくまで経験的法則であり、すべての産業・製品に当てはまるわけではない。
- 破壊的技術革新によって経験曲線がリセットされる可能性がある。
- コスト削減至上主義による品質劣化や、競合他社の同時コスト削減によって優位が相殺されるリスク。

## 3. 習熟の認知メカニズム

### 3.1 Anderson の ACT-R モデル

John Anderson の ACT-R (Adaptive Control of Thought — Rational) は人間の認知を**宣言的記憶 (Declarative Memory)** と**手続き的記憶 (Procedural Memory)** の二系統で説明する。

| 記憶種類 | 内容 | 習熟との関係 |
|---|---|---|
| 宣言的記憶 | 事実・概念の知識 (「〜とは何か」) | 学習初期に意識的に参照 |
| 手続き的記憶 | 行動・スキルの知識 (「〜のやり方」) | 習熟とともに自動化・高速化 |

練習により宣言的な知識が手続き的な知識へコンパイルされ、意識的処理から自動処理へ移行する — これが「コツを掴む」の認知的実体である。

### 3.2 Fitts & Posner の 3 段階モデル (1967)

| 段階 | 名称 | 特徴 |
|---|---|---|
| 1 | 認知段階 (Cognitive Phase) | 手順を言語的に理解しながら試行錯誤。エラー多い。 |
| 2 | 連合段階 (Associative Phase) | 運動パターンが形成され始め、エラーが減少。 |
| 3 | 自律段階 (Autonomous Phase) | 動作が自動化し、注意なしに正確・高速に実行可能。 |

[05_暗黙知と技能伝承](./05_暗黙知と技能伝承.md) の Dreyfus 段階モデルと対応関係にある。

### 3.3 Power Law of Practice

Newell & Rosenbloom (1981) は「練習回数とパフォーマンス (応答時間・エラー率) の対数が線形関係にある」という **練習の冪乗則 (Power Law of Practice)** を示した。

```
RT = a + b · N^(-c)

RT: 反応時間, N: 練習回数, a/b/c: 定数
```

これは個人レベルの習熟速度の普遍的なパターンを記述し、学習曲線の認知的根拠を与える。

## 4. 忘却・中断と習熟の保持

### 4.1 忘却曲線 (Ebbinghaus, 1885)

Hermann Ebbinghaus は無意味綴り (nonsense syllables) の記憶実験から、時間経過に伴う記憶減衰が対数関数に従うことを示した。

```
R = e^(-t/S)

R: 保持率, t: 経過時間, S: 記憶の強さ
```

製造業では、まれにしか実施しない工程 (定期メンテナンス手順、年次切替作業) での習熟低下リスクが高い。

### 4.2 分散練習効果 (Spacing Effect)

同一時間の練習でも、集中練習より分散練習 (インターバルをおいた反復) のほうが長期保持に優れる。生産訓練スケジュールの設計に直接適用できる原則。

## 5. 組織的学習と知識移転

### 5.1 工場レベルの学習

Argote & Epple (1990) は Liberty Ship (第二次大戦の輸送船) の大量建造データを分析し、**工場ごとに異なる学習率が観察され、人員交替・生産中断が学習の喪失 (Depreciation) を引き起こす**ことを実証した。

主な知見:
- 知識は人員・設備・ルーティンに体化されており、人員退職や設備入替でコストが上昇する。
- 同系列工場間では知識移転が可能だが、移転には積極的な努力が必要。

### 5.2 知識の腐食 (Knowledge Depreciation)

習熟によって獲得されたコスト優位は、以下の要因で失われる:

| 要因 | 内容 |
|---|---|
| 人員交替 | 熟練者の退職・異動による経験の流出 |
| 生産中断 | 長期停止後の再起動で手順・感覚が劣化 |
| 製品モデルチェンジ | 新設計により習熟がリセットされる |
| 設備更新 | 新設備の操作習熟に時間がかかる |

## 6. 批判と限界

### 6.1 Wright 80% 法則のデータ依存性批判

Wright の学習曲線は航空機製造という特定の産業・時代の経験的観察から導出されており、**普遍的法則としての妥当性には限界がある**。学習率 (80% や 85%) は産業・製品・工程によって大きく異なり、事後的なフィッティングで得られる数値に過ぎないという批判がある (Yelle, 1979)。また、曲線を描くためには十分な生産量データが必要であり、少量生産・多品種製品では統計的に信頼できる学習率の推定が困難である。本アプリが対象とする製造現場でも、品種切り替えが頻繁であれば学習曲線モデルの適用精度は低くなる。

### 6.2 BCG 経験曲線の「過剰戦略化」批判

Ghemawat (1985) は BCG の経験曲線に基づく「市場シェアを最大化すれば必ずコスト優位が得られる」という戦略処方に対し、**シェアとコストの因果関係は双方向であり、シェア獲得がコスト低下を自動的に保証しない**と批判した。経験曲線は相関を示すが、その因果メカニズム (どのような管理行動が習熟を促進するか) を説明しないため、「経験を積む」という行為への過剰な投資を誘発するリスクがある。製造現場においても、単に生産量を増やすだけでは習熟は進まず、意図的な訓練・標準化・フィードバックが必要であることを見落とさないことが重要である。

### 6.3 学習曲線の非連続性 — プラトーと後退現象

現実の習熟は単調な減衰曲線をたどらず、**学習の停滞 (プラトー)** や一時的な後退が生じることがある。特に Fitts & Posner の段階モデルで「連合段階」から「自律段階」への移行期には、意識的な制御から自動処理への切り替えがうまくいかず、一時的にパフォーマンスが低下する現象 (Crossover Dip) が報告されている。製造手順の変更直後に経験豊富な作業者のエラー率が上昇するのは、この非連続性の現れである。

### 6.4 知識腐食とカウンターバランスの困難

Argote & Epple (1990) は工場レベルの知識腐食 (Knowledge Depreciation) を実証したが、**腐食速度の定量化は難しく**、どの程度の中断・人員交替で習熟がリセットされるかを事前に予測する汎用モデルは存在しない。特に定期メンテナンスや年次切替のように稼働頻度の低い工程では、「中断→腐食」のサイクルが反復し、習熟の蓄積が難しい。Skill Decay 対策として分散練習 (スペーシング効果) が推奨されるが、製造現場での計画的な復習スケジュールの設計・実施は組織的コストが高い。

### 6.5 本アプリへの含意

習熟曲線の知見は、ナビゲーションアプリのアダプティブ UI 設計に活用できる。新人には詳細な手順・画像・動画を提示し、習熟が進むにつれて提示情報を段階的に削減する Fading 機能は、Power Law of Practice を工学的に実装したものである。ただし、プラトーや後退現象を考慮し、「習熟済み」の作業者でも稀な工程や長期ブランクがある場合には詳細ガイドを復元するアラート設計が必要である。
