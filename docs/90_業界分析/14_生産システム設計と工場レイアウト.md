# 生産システム設計と工場レイアウト

## 1. 生産システムの類型

製造業の生産システムは、製品の種類と数量の関係によって以下のように類型化される。

| 類型 | 特徴 | 例 |
|---|---|---|
| プロジェクト型 (Project) | 1 品、固定場所、複数工程が品に集まる | 造船、プラント建設 |
| ジョブショップ (Job Shop) | 多品種少量、機能別レイアウト、複雑な工程ルーティング | 金型、試作品 |
| バッチ (Batch) | 中品種中量、ロット単位の生産 | 部品加工、食品 |
| フローショップ (Flow Shop) | 少品種大量、工程順にレイアウト | 家電組立 |
| 連続生産 (Continuous) | 単品種、24 時間連続 | 石油精製、製紙 |

Hayes & Wheelwright (1979) の**製品-プロセスマトリックス**は、製品のライフサイクル段階とプロセスの類型を 2 軸として戦略的整合性を評価する枠組みを提供した。対角線上の配置が競争優位を生み、対角からの逸脱はコスト・柔軟性の非効率を招く。

## 2. 工場レイアウトの基本形式

| レイアウト | 配置原理 | 長所 | 短所 |
|---|---|---|---|
| 機能別 (Process Layout) | 同種機械を集めた部門配置 | 柔軟性高い、稼働率向上 | 搬送距離長い、WIP 増加 |
| 製品別 (Product Layout) | 工程順に機械を並べたライン | 流れが速い、WIP 少ない | 設備専用化、故障に脆弱 |
| セル型 (Cellular Layout) | 部品族ごとに必要工程を集約 | 段取り削減、小集団管理 | 機械重複投資 |
| 固定位置型 (Fixed Position) | 製品を動かさず工程を集める | 大型品・重量品に適合 | スペース・調整が困難 |

## 3. Muther の SLP (Systematic Layout Planning)

Richard Muther が 1961 年に体系化した、工場・倉庫・事務所等のレイアウトを計画的に決定する汎用手順。

### 3.1 SLP のステップ

```
1. P-Q 分析 (Product-Quantity Analysis)
   ↓ 品目×数量マトリックスで主要品目を特定
2. 工程流れ分析 (Flow Analysis)
   ↓ From-To チャートで部門間の流れを定量化
3. アクティビティ関連分析 (Activity Relationship)
   ↓ REL チャートで非物流的な近接必要性を評価
4. 関連ダイアグラム (Relationship Diagram)
   ↓ 流れ+関連を視覚化 (バブルダイアグラム)
5. スペース要件の決定 (Space Determination)
   ↓ 機械・通路・保管スペースの面積算出
6. スペースレイアウト (Space Relationship Diagram)
   ↓ 面積を割り付けた詳細レイアウト案の作成
7. 制約との照合と選定 (Modification and Evaluation)
```

### 3.2 REL チャートの評価コード

| コード | 意味 |
|---|---|
| A (Absolutely Necessary) | 絶対に隣接が必要 |
| E (Especially Important) | 特に重要 |
| I (Important) | 重要 |
| O (Ordinary) | 通常 |
| U (Unimportant) | 重要でない |
| X (Undesirable) | 隣接不可 |

## 4. GT (Group Technology) とセル生産

**GT (Group Technology)** は、多品種加工製品の中から類似の形状・材質・加工手順を持つ「部品族 (Part Family)」を識別し、それらを同一設備グループで処理することで段取り削減と効率化を図る概念。Mitrofanov (1966) がソ連で、その後 Ham (1976) が西側で普及させた。

### 4.1 部品族の形成方法

| 方法 | 内容 |
|---|---|
| 視覚検査 | 経験者が図面を目視分類 |
| 部品分類・コーディング | Opitz、MICLASS 等のコードシステムで部品の特徴を数値化 |
| 生産フロー分析 (PFA) | 工程経路の類似性に基づくクラスタリング |

### 4.2 製造セル (Manufacturing Cell) の設計

部品族に対応する機械・設備のグループ。旋盤・フライス・ボール盤等を 1 セルに配置し、1 部品族を一貫して加工する。

利点:
- **段取り削減**: 同族部品は治具・工具の共有率が高く、切替時間が短い
- **運搬短縮**: セル内で工程が完結し、部門間搬送が減少
- **品質の可視化**: 小規模で問題が見えやすい
- **多能工の育成**: セル内の複数工程を習熟する必要が生まれる

## 5. ラインバランシング

**ラインバランシング (Assembly Line Balancing: ALB)** は、製品の組立作業を複数の作業ステーションに割り付け、稼働率を最大化 (ステーション間の負荷を均等化) する問題。

### 5.1 基本指標

```
サイクルタイム (Cycle Time) = 生産可能時間 / 必要生産数

最少ステーション数 = ⌈全作業時間 / サイクルタイム⌉

ラインバランス効率 = 全作業時間 / (ステーション数 × サイクルタイム)
```

### 5.2 主なヒューリスティック手法

| 手法 | 割付優先順位の基準 |
|---|---|
| 最長作業時間優先 (Helgeson & Birnie, 1961) | 作業時間が最長のタスクを優先割付 |
| 後続タスク数優先 | 後続に多くのタスクを持つものを優先 |
| 最長経路優先 (RPW: Ranked Positional Weight) | タスク自身と後続タスクの時間合計で優先付け |

厳密解を求める Integer Programming も使用されるが、NP 困難問題のためヒューリスティックが主流。

## 6. マスカスタマイゼーション

B. J. Pine II *Mass Customization* (1993) が提唱した「大量生産のコストで個別品を届ける」製造戦略。

### 6.1 モジュール設計

製品をモジュール (交換可能な機能ブロック) に分解し、組み合わせによるバリエーションを実現する。プラットフォームと派生モデルの管理が鍵。

### 6.2 遅延差別化 (Postponement)

製品を汎用品として製造しておき、顧客注文確定後に最終仕様化 (色付け・言語切替・オプション追加) を行う戦略。在庫リスクの削減と顧客対応の柔軟性を両立させる。
