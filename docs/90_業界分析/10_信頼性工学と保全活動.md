# 信頼性工学と保全活動

## 1. 信頼性工学の起源

**信頼性工学 (Reliability Engineering)** は第二次世界大戦後から冷戦期の米国において、電子兵器・宇宙機器の信頼性問題に対処する必要から体系化された。

- **AGREE 報告書 (1957)**: 米国防省 Advisory Group on Reliability of Electronic Equipment が発行。軍用電子機器の信頼性保証プロセスを初めて体系化した。
- **NASA の宇宙開発 (1960〜70 年代)**: Apollo 計画での FMEA・FTA の実用化。
- **MIL-HDBK-217 (1965)**: 電子部品の信頼性予測ハンドブック。
- 民間では原子力・航空産業が牽引し、IEC 60300 シリーズとして国際標準化が進んだ。

## 2. 信頼性の基本指標

### 2.1 信頼度関数 R(t)

アイテムが時刻 0 から時刻 t まで故障なく動作する確率。

```
R(t) = P(T > t)    T: 故障発生時刻の確率変数
```

### 2.2 故障率と MTBF

| 指標 | 定義 | 活用 |
|---|---|---|
| 故障率 λ(t) | 単位時間あたりの故障発生率 | 偶発故障期では定数として扱う |
| MTBF (Mean Time Between Failures) | 修復可能なアイテムの平均故障間隔 | 可用性・保全計画の設計 |
| MTTF (Mean Time To Failure) | 修復不可能なアイテムの平均寿命 | 交換周期設計 |
| MTTR (Mean Time To Repair) | 平均修復時間 | 保全資源の計画 |

### 2.3 アベイラビリティ (Availability)

要求された時間に対してアイテムが機能を果たせる状態にある割合。

```
固有アベイラビリティ Ai = MTBF / (MTBF + MTTR)
```

## 3. バスタブ曲線

製品ライフサイクルを通じた故障率の典型的なパターンを示す。

```
故障率
  │  \               / 摩耗故障期
  │   \             /  (Wear-out Failure)
  │    \___________/
  │  初期故障期    偶発故障期
  │  (Early Failure) (Random Failure)
  └──────────────────────────── 時間
```

| 期間 | 原因 | 対策 |
|---|---|---|
| 初期故障期 | 設計・製造・材料の欠陥 | バーンイン (Burn-in)、スクリーニング |
| 偶発故障期 | ランダムな外的ストレス | 冗長設計、フェイルセーフ |
| 摩耗故障期 | 疲労・摩耗・腐食 | 予防交換、オーバーホール |

## 4. ワイブル分布によるライフモデリング

Waloddi Weibull (1951) が提案した確率分布。形状パラメータ (β) で故障率の傾向を表現できる汎用性の高い分布。

```
信頼度関数: R(t) = exp[-(t/η)^β]

β < 1: 故障率が時間とともに減少 (初期故障期)
β = 1: 故障率が一定 (偶発故障期、指数分布と一致)
β > 1: 故障率が時間とともに増加 (摩耗故障期)
```

ワイブル確率紙へのプロットで形状・尺度パラメータを推定し、保全周期や保証期間の設計に用いる。

## 5. 故障解析手法

### 5.1 FTA (Fault Tree Analysis)

最上位の「望ましくない事象 (Top Event)」から論理ゲート (AND/OR) を用いて故障の根本原因まで下降的に展開する演繹的手法。最小カットセット (MCS) の算出が可能。

### 5.2 ETA (Event Tree Analysis)

起因事象 (Initiating Event) から一連の安全機能の成功/失敗を帰納的に展開し、各シナリオの発生確率を算出する手法。PRA (Probabilistic Risk Assessment) の主要ツール。

## 6. 保全戦略の分類

| 戦略 | 略号 | 概要 |
|---|---|---|
| 事後保全 | BM (Breakdown Maintenance) | 故障してから修理 |
| 時間基準予防保全 | TBM (Time-Based Maintenance) | 使用時間・カレンダー周期で定期保全 |
| 状態基準予防保全 | CBM (Condition-Based Maintenance) | 振動・油分析等で状態を監視し保全 |
| 予知保全 | PdM (Predictive Maintenance) | CBM の高度版; 劣化予測で最適タイミングで保全 |
| 信頼性中心保全 | RCM (Reliability-Centered Maintenance) | 故障モードの分析に基づき戦略を決定 |

### 6.1 RCM (Reliability-Centered Maintenance)

Nowlan & Heap が 1978 年に United Airlines 向けに開発した手法 (後に MSG-3 として航空業界標準化)。各機能・故障モードに対し「この故障を予防する価値があるか」を問い、BM / TBM / CBM / 設計変更を最適配分する。IEC 60300-3-11 に規定。

RCM 意思決定プロセス (7 つの問い):
1. 対象の機能と性能基準は何か
2. 機能不全はどのように生じるか
3. 機能不全の原因 (故障モード) は何か
4. 各故障モードはどのような影響をもたらすか
5. 影響はどの点で重大か
6. 機能不全を予防・検出するには何ができるか
7. 適切な保全作業がない場合はどうするか

## 7. TPM (Total Productive Maintenance)

中嶋清一が日本プラントメンテナンス協会 (JIPM) の前身で 1971 年に体系化した全員参加型の生産保全活動。1988 年に JIPM は TPM 優秀賞を設立し、世界 50 カ国以上に普及した。

### 7.1 TPM の定義 (JIPM)

1. 設備総合効率 (OEE) の極大化を目標とする
2. PM (予防保全・生産保全・保全予防) の体制確立
3. 設備の計画から廃棄まで全ライフサイクルにわたる
4. 生産・保全・設計の全部門が参加
5. 小集団活動を通じた自主保全の徹底

### 7.2 TPM 8 本柱

| 柱 | 内容 |
|---|---|
| ① 個別改善 | ロスを徹底的に排除し、設備・プロセスを最適化 |
| ② 自主保全 | オペレーター自身が清掃・点検・給油・増し締めを実施 |
| ③ 計画保全 | 保全部門がゼロ故障に向けた計画的保全を実施 |
| ④ 教育・訓練 | オペレーター・保全員のスキルアップ |
| ⑤ 初期管理 | 設備・製品の初期流動管理と MP (Maintenance Prevention) 設計 |
| ⑥ 品質保全 | 不良ゼロを目指し、品質に影響する設備条件を管理 |
| ⑦ 管理間接 | 間接部門・事務部門のムダ排除と効率化 |
| ⑧ 安全・環境 | ゼロ災害・環境保全の達成 |

### 7.3 OEE と 8 大ロス / 16 大ロス

**OEE (Overall Equipment Effectiveness)** は設備の総合的な効率を示す KPI ([07_スマートファクトリー](./07_スマートファクトリーと作業のデジタル化.md) も参照)。

TPM では OEE の阻害要因として **8 大ロス** を定義する:

| 分類 | ロス |
|---|---|
| 停止ロス | 故障ロス、段取り・調整ロス |
| 性能ロス | 空転・チョコ停ロス、速度低下ロス |
| 不良ロス | 工程不良ロス、初期収率ロス |
| その他 | 管理ロス、動作ロス (オペレーター) 等 |

さらに人・エネルギー・型治工具ロスを含む **16 大ロス** も定義される。

### 7.4 自主保全 7 ステップ

オペレーター主体で設備を守る自主保全活動の段階的展開。

| ステップ | 活動 |
|---|---|
| 1 | 初期清掃 (汚れを落とし、不具合を発見) |
| 2 | 発生源・困難個所対策 (汚れが出ない、清掃しやすい状態へ) |
| 3 | 清掃・点検・給油基準書の作成 |
| 4 | 総点検 (機構を理解し、潜在欠陥を摘出) |
| 5 | 自主点検 (自主点検チェックシートで維持管理) |
| 6 | 標準化 (管理標準・作業標準の整備) |
| 7 | 自主管理の徹底 (日常管理・改善活動の自律化) |
