# 認知工学と状況認識

## 1. 認知工学の定義

**認知工学 (Cognitive Engineering)** は David Woods と Erik Hollnagel が 1980 年代に提唱した学際的領域であり、人間が複雑で動的な環境の中でどのように機能するかを理解し、システム設計に活かすことを目的とする。古典的な「人間の誤りを補正するシステム設計」から、「人間とシステムの協働による適応的な問題解決」へと視点が転換している。

認知工学の特徴:
- 実験室ではなく実際の現場 (制御室、手術室、工場フロア) を研究の場とする。
- 課題の複雑さと人間の認知能力の相互作用に焦点を当てる。
- エラー防止より「うまくいっている状態」の理解を重視する流れも含む。

## 2. 状況認識 (Situation Awareness: SA)

**状況認識** は Mica Endsley (1988, 1995) が提唱した概念であり、「ある時空間内の環境要素の知覚、その意味の理解、および近未来の状態の予測」と定義される。

### 2.1 Endsley の 3 層 SA モデル

```
レベル 3: 予測 (Projection)
   「この状態が続けば 5 分後はどうなるか」
         ↑ (理解をもとに将来状態を外挿)
レベル 2: 理解 (Comprehension)
   「この状態は何を意味するのか」
         ↑ (知覚した要素を統合・解釈)
レベル 1: 知覚 (Perception)
   「何が起きているか」
   (システムの状態・属性・動態の知覚)
```

レベル 1 の失敗 (見逃し、見間違い) は最も頻度が高いが、レベル 2・3 の失敗 (理解の誤り、不適切な予測) がより深刻な事態に繋がりやすい。

### 2.2 SA 喪失の原因

Endsley (1995) は SA Error Taxonomy を提案し、SA 喪失の主要原因を以下に分類した。

| カテゴリ | 内容 | 例 |
|---|---|---|
| データの利用不可 | 必要な情報がシステムから得られない | 死角、センサ未設置 |
| データの見逃し | 情報はあるが注意が向かなかった | 警報の見落とし |
| 記憶の失敗 | 取得した情報を忘れた | 多重タスク下での情報保持の失敗 |
| 誤ったメンタルモデル | 状況の解釈フレームが不正確 | 設備動作の誤解 |
| 高すぎる負荷 | 認知容量を超えた情報量 | 緊急対応時の判断マヒ |

### 2.3 SA の測定: SAGAT

**SAGAT (Situation Awareness Global Assessment Technique)** は訓練シミュレーション中に突然停止し、オペレーターに現状認識に関する質問に答えさせる直接測定技法 (Endsley, 1988)。他に SART (Situation Awareness Rating Technique) などの主観評定法もある。

## 3. 自然主義的意思決定 (NDM: Naturalistic Decision Making)

Gary Klein らが 1980 年代末から展開した研究領域。実験室の「選択肢比較」モデルではなく、実際の現場 (消防、看護、軍事、製造) でのリアルタイム意思決定を研究する。

### 3.1 認識誘導型意思決定 (RPD: Recognition-Primed Decision)

Klein の代表的モデル。熟練者はオプションを列挙・比較するのではなく、状況の特徴パターンを認識し、最初に思い浮かんだ行動候補を精神的シミュレーションで検証してそのまま実行するか修正するかを判断する。

```
状況を知覚
    ↓
パターン認識 ← 経験の蓄積
    ↓
最初の行動候補が浮かぶ
    ↓
精神的シミュレーション (これで問題ないか?)
    ↓ Yes                   ↓ No (問題発見)
   実行                  修正した行動候補を試す
```

この研究は「真の意思決定は選択ではなく認識から始まる」という視点を示し、熟練者の「勘」や「コツ」に認知科学的説明を与えた ([05_暗黙知と技能伝承](./05_暗黙知と技能伝承.md) の Dreyfus モデルと対応)。

### 3.2 NDM の製造現場への含意

- 熟練オペレーターは手順書をステップごとに読むのではなく、状況の「全体像」で判断する。
- 訓練設計では選択肢の暗記より「状況パターン認識力の育成」が重要。
- 異常時対応の意思決定速度は「経験の深さ = 蓄積されたパターン数」に依存する。

## 4. 分散認知 (Distributed Cognition)

Edwin Hutchins が *Cognition in the Wild* (1995) において提唱した認知理論。「認知は個人の頭の中だけに存在するのではなく、人・道具・表象・環境にまたがって分散している」と主張する。

研究の出発点は海軍艦船の航法チームの観察。航路計算は単一の人間の頭の中でなく、チームの会話・海図・計算器・ルーター等の表象媒体を通じて「システム全体」として遂行されていた。

製造への含意:

| 分散認知の要素 | 製造現場での具体例 |
|---|---|
| 表象 (Representation) | 作業指示書、管理板、アンドン |
| 道具 (Artifact) | 治具、ゲージ、タブレット端末 |
| 社会的調整 (Social Coordination) | 引継ぎ連絡、朝礼、異常時の呼び出し |
| 環境のアフォーダンス | 部品棚のラベリング、位置の符号化 |

作業者が環境に「外部化された認知」に依存していることを踏まえると、5S やデジタル化は単なる整理整頓ではなく「認知インフラの設計」である。

## 5. 連合認知システム論 (Joint Cognitive Systems)

Hollnagel & Woods (2005) は人間とシステムを「連合認知システム (Joint Cognitive Systems: JCS)」として一体的に捉えることを提唱した。「オートメーション vs 人間」の対立ではなく、双方が互いの限界を補い合う設計が鍵となる。
