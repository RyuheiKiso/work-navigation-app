# 作業の定義と分類

## 1. 「作業」の語義

英語圏では製造現場の文脈において "work"・"operation"・"task" の三語が使い分けられる。

| 英語 | 典型的な粒度 | 例 |
|---|---|---|
| work | 広義の労働・仕事一般 | 作業員の日常的な労務 |
| operation | 生産工程上の処理単位 | 「溶接工程」「組立工程」 |
| task | 人間が遂行する目標指向的な行為単位 | 「ボルト締め付け」「外観検査」 |

日本語「作業」は operation と task の両義を含む。文脈に応じた粒度の明示が重要である。

## 2. JIS Z 8141 による定義

JIS Z 8141:2001「生産管理用語」は生産管理の標準用語を定める。関連する定義を以下に示す。

- **工程 (process)**: 原材料を製品に変換するための系統的な一連の活動。
- **作業 (operation)**: ある目的のために、人・機械などが行う一定のまとまりのある活動。
- **要素作業 (work element)**: 作業を構成する最小の単位であり、これ以上分割すると意味をなさなくなる作業の構成要素。

標準における「作業」はあくまで「目的をもつ活動の単位」として中立的に定義されており、人間が行うか機械が行うかを問わない。

## 3. IE における階層的分類

Industrial Engineering (生産工学) では作業を次の階層構造で捉える。

```
工程 (Process)
  └─ 単位作業 (Unit Operation)
        └─ 要素作業 (Work Element)
              └─ 動素 (Therblig)
```

| 階層 | 内容 | 分析ツール |
|---|---|---|
| 工程 | 素材が機能的に変化する段階 (加工・検査・搬送・保管) | 工程分析、工程図 |
| 単位作業 | 作業者・機械単体で完結する一連の動作 | 作業分析 |
| 要素作業 | 計測可能な最小の有意義な動作単位 | 時間研究 (stopwatch study) |
| 動素 (サーブリッグ) | F.B. Gilbreth が定義した 18 種の基本動作要素 | 動作研究、micromotion study |

### 18 動素 (Therblig)

Gilbreth 夫妻が提唱した基本動作要素であり、どの製造工程においても人手作業はこれらの組み合わせとして分解できるとされる。代表的なものを示す。

| 記号 | 名称 | 概要 |
|---|---|---|
| Te | 手を伸ばす (Transport Empty) | 物体に向けて空手で手を動かす |
| G | つかむ (Grasp) | 対象物を把持する |
| TL | 運ぶ (Transport Loaded) | 物体を持ちながら移動する |
| P | 位置決め (Position) | 使用・組付けに合わせて位置を定める |
| RL | 離す (Release Load) | 対象物を放す |
| I | 検査する (Inspect) | 品質・数量を確認する |
| U | 使用する (Use) | 工具・機械を操作する |
| D | 遅延 (Delay) | 不可避の待機 |

## 4. 作業時間の分類

IE では作業に費やす時間を以下の区分で分析する。

```
総時間
├── 正味作業時間
│   ├── 主作業 (Main Operation)        ← 製品価値を直接生む活動
│   └── 付随作業 (Incidental Operation) ← 主作業に付随する補助的活動
└── 余裕時間
    ├── 管理余裕 (Management Allowance)
    ├── 疲労余裕 (Fatigue Allowance)
    └── 個人余裕 (Personal Allowance)
```

さらに工場全体での稼働分析では「付帯作業」(段取り、清掃等) を別途計上するケースもある。

## 5. 4M / 5M+1E フレーム

製造工程における変動要因を体系化した管理フレームワーク。

| 要素 | 内容 |
|---|---|
| Man (人) | 作業者のスキル・経験・状態 |
| Machine (機械) | 設備の種類・状態・精度 |
| Material (材料) | 原材料・部品の品質・ロット |
| Method (方法) | 作業手順・標準・条件 |
| (Measurement) | 計測・検査方法 |
| (Environment) | 温度・湿度・照明・騒音 |

異常・品質問題の原因分析 (特性要因図) の大骨として広く用いられる。
