# 暗黙知と技能伝承

## 1. 暗黙知の概念 — Michael Polanyi

Michael Polanyi (1891–1976) は *The Tacit Dimension* (1966) において、人間の知識は「言語化・明示化できるものより多くを知っている」という洞察から出発し、二種類の知識を区別した。

| 種類 | 内容 | 例 |
|---|---|---|
| 形式知 (Explicit Knowledge) | 言語・記号・数式で表現・伝達できる知識 | 作業手順書、技術仕様書 |
| 暗黙知 (Tacit Knowledge) | 言語化が困難で、実践の中に体化されている知識 | 熟練工の加工感覚、職人の目利き |

Polanyi の命題「われわれは語れる以上のことを知っている (We can know more than we can tell)」は、暗黙知が知識論的に形式知に先行し、すべての形式知は暗黙知に根ざすことを含意する。

### 1.1 焦点的認識と補助的認識

Polanyi は知覚の構造を「補助的認識 (subsidiary awareness)」と「焦点的認識 (focal awareness)」の区分で説明する。熟練者は個々の動作細部を補助的に意識しながら全体の課題に焦点を当てる。この統合された知覚様式が暗黙知の本質である。

## 2. 野中郁次郎の SECI モデル

野中郁次郎・竹内弘高は *The Knowledge-Creating Company* (1995) において、組織的知識創造のメカニズムを**暗黙知と形式知の相互変換**として定式化し、4 つのモードからなる SECI モデルを提唱した。

```
                  暗黙知 → 暗黙知         暗黙知 → 形式知
                ┌─────────────────┬─────────────────┐
                │ 共同化            │ 表出化            │
                │ (Socialization)   │ (Externalization) │
                │ 体験・模倣・観察  │ メタファー・概念化 │
                ├─────────────────┼─────────────────┤
                │ 内面化            │ 連結化            │
                │ (Internalization) │ (Combination)     │
                │ 実践・体化        │ 体系化・編集      │
                └─────────────────┴─────────────────┘
                  形式知 → 暗黙知         形式知 → 形式知
```

| モード | 変換 | 主なメカニズム |
|---|---|---|
| 共同化 (Socialization) | 暗黙知→暗黙知 | OJT、見習い、同一空間での共同作業 |
| 表出化 (Externalization) | 暗黙知→形式知 | ダイアローグ、メタファー、手順書化 |
| 連結化 (Combination) | 形式知→形式知 | データベース化、知識ベースの構築 |
| 内面化 (Internalization) | 形式知→暗黙知 | 反復実践、ドキュメント学習、シミュレーション |

SECI サイクルは個人→集団→組織と螺旋的に拡大 (Knowledge Spiral) し、組織の競争優位を支える継続的な知識創造を実現する。

## 3. D. A. Schön の省察的実践家

Donald Schön は *The Reflective Practitioner* (1983) において、実践の中の知識形成を「行為の中の省察 (Reflection-in-Action)」として論じた。熟練した実践家は規則に従って演繹的に行動するのではなく、状況との「対話 (conversation with the situation)」を通じて知識を動的に生成する。この視点は、作業現場での「コツ」や「カン」が決して非合理ではなく、洗練された状況知識の表れであることを示す。

## 4. 技能の段階的習熟 — Dreyfus モデル

Hubert Dreyfus & Stuart Dreyfus は *Mind over Machine* (1986) において、人工知能との対比から、人間の技能習得を 5 段階で記述した。

| 段階 | 名称 | 特徴 |
|---|---|---|
| 1 | 初心者 (Novice) | コンテキスト非依存のルール適用; 状況全体を把握できない |
| 2 | 上達者 (Advanced Beginner) | 状況の類型 (aspecto) を認識; まだルール依存 |
| 3 | 一人前 (Competent) | 多くの情報から重要な要素を選択; 結果に責任感 |
| 4 | 熟達者 (Proficient) | 状況を直感的に把握; 行動は分析的選択 |
| 5 | 達人 (Expert) | 状況・行動が一体化; 「なぜそうするか」が言語化困難 |

ルールベースの段階 (1〜3) から直観ベースの段階 (4〜5) への移行が、暗黙知の深化に対応する。

## 5. 認知的徒弟制

Collins, Brown & Newman は *Cognitive Apprenticeship* (1989) において、中世ギルドの徒弟制を認知的スキルの習得に援用した。

主要な教授法的要素:

| 要素 | 内容 |
|---|---|
| モデリング (Modeling) | 専門家が自身の思考過程を声に出しながら作業を見せる |
| コーチング (Coaching) | 学習者の実践を観察し、フィードバックを与える |
| スキャフォールディング (Scaffolding) | 学習者の能力を超える部分を一時的に支援し、徐々に外す |
| フェーディング (Fading) | 支援を段階的に撤退し、自律性を育む |
| 明示化 (Articulation) | 学習者に知識・判断・問題解決過程を言語化させる |
| 省察 (Reflection) | 専門家のパフォーマンスと自身のそれを比較分析させる |

## 6. 製造業における技能継承問題

### 6.1 2007 年問題と団塊世代退職

日本では戦後ベビーブームで生まれた「団塊の世代」(1947〜49 年生まれ) が 2007 年前後に定年退職を迎えた。製造業各社では熟練工の大量退職に伴い、長年の OJT で培われた暗黙知・技能が一斉に失われるリスクが現実となった。

### 6.2 技能継承の構造的困難

- **暗黙知の言語化困難**: 「感覚」「コツ」は言葉にしづらく、映像・センサデータ等の補完が必要。
- **OJT の時間的制約**: 習熟には数年〜十数年を要し、退職前に完全な移転が困難。
- **多能工化のトレードオフ**: 広浅の知識習得と専門深掘りのバランス。
- **暗黙知の個人依存**: 組織ではなく特定個人に知識が偏在する。

### 6.3 対応アプローチ

| アプローチ | 手段 |
|---|---|
| 形式知化 | 技能マニュアル、作業動画、チェックリスト作成 |
| センシング | 加工音・振動・熱・力覚のデータ収集と AI 学習 |
| 認知的徒弟制の制度化 | メンタリング制度、ペア作業の長期設計 |
| シミュレーション | VR/AR を用いた疑似的な実践環境の構築 |
| ナレッジマネジメント | SECI サイクルを意識した知識共有プロセスの設計 |
