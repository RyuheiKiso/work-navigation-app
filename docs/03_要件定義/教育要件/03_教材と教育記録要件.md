# 03 教材と教育記録要件

本書の責務は、本システムの教育実施に使用する教材の種類・作成要件・品質基準および教育実施記録の保存要件・評価実施要件を確定することである。教材の品質と記録の完全性が教育効果の検証可能性を担保する。

---

## 1. 教材の種類と作成要件

### 1-1. 教材の 4 種類

EDU-027: 本システムの教育に使用する教材を以下の 4 種類に確定する。

| 教材種別 | 目的 | 主要対象者 |
|---|---|---|
| 操作マニュアル | 全機能の体系的な操作説明 | 全対象者（対象者別に内容を分ける） |
| 動画チュートリアル | 視覚的な操作手順の提示 | 作業員・外国人実習生 |
| ハンズオン演習シナリオ | 実際の操作を通じた習熟 | 全対象者（ハンズオン演習で使用） |
| QRC（クイックリファレンスカード） | 現場での即席参照 | 作業員・外国人実習生 |

### 1-2. 操作マニュアルの作成要件

EDU-028: 操作マニュアルは対象者ロール別に 4 種類を作成する。

| マニュアル種別 | 対象ロール | 必須内容 |
|---|---|---|
| 作業員向け操作マニュアル | operator | タブレット APP の全操作・電子サイン・不適合起票・Kaizen Teian 起票・オフラインモード対応 |
| 現場監督向け操作マニュアル | supervisor | 作業員向け全操作に加え、アンドン対応・CAPA 管理・認定登録・SOP レビュー |
| マスタ編集者向け操作マニュアル | master_admin | 現場監督向け全操作に加え、SOP 改訂フロー・DSL 設定・Publish・廃止処理 |
| IT 担当向け操作マニュアル | system_admin | システム構築・バックアップ・パッチ適用・ユーザー管理・障害対応手順 |

EDU-029: 操作マニュアルの品質基準を以下の通り確定する。

| 品質基準 | 内容 |
|---|---|
| スクリーンショット必須 | 主要な操作画面のスクリーンショットを操作手順ごとに挿入する |
| 版数管理 | 版数（v1.0.0 形式）・更新日・更新内容を表紙に記載する |
| SOP 改訂との連動 | 本システムの機能変更時は操作マニュアルを 30 日以内に更新する |
| 多言語版の提供 | 外国人実習生対応要件（EDU-020）に従い、英語版を必須で提供する |

### 1-3. 動画チュートリアルの作成要件

EDU-030: 動画チュートリアルは以下の仕様で作成する。

| 仕様項目 | 内容 |
|---|---|
| 動画の尺 | 1 本あたり 3 分以内。長い操作は分割して複数本に分ける |
| 字幕 | 日本語字幕を必須とする。英語字幕は外国人実習生が在籍する工場では必須とする |
| ナレーション | やさしい日本語のナレーションを付与する（EDU-017 の設計原則に従う） |
| 保存形式 | MP4 形式・1280×720 px 以上。社内ファイルサーバーに保存し、本システムの教材リンクから参照する |

### 1-4. ハンズオン演習シナリオの作成要件

EDU-031: ハンズオン演習シナリオは以下のシナリオを対象者別に作成する。

| シナリオ | 対象者 | 演習内容 |
|---|---|---|
| 標準作業の SOP ナビゲーション | 作業員 | 代表工程 1 件を最初から最後まで操作する |
| 不適合起票シナリオ | 作業員・現場監督 | ダミーの不適合事例に基づいて起票・写真添付・提出を行う |
| SOP 改訂シナリオ | マスタ編集者 | ダミー SOP の Step 1 件を変更・承認・Publish する |
| 障害対応シナリオ | IT 担当 | ステージング環境でサービス停止を意図的に発生させ、切り分け・復旧を行う |

**本節で確定した方針**
- 操作マニュアル・動画チュートリアル・ハンズオン演習シナリオ・QRC の 4 種を教材として確定する。
- 操作マニュアルの版数管理・スクリーンショット必須・30 日以内更新・英語版必須を品質基準として確定する。
- 動画は 3 分以内・字幕必須・MP4 形式の仕様を確定する。

## 2. 教育実施記録の要件

### 2-1. 記録の必須項目

EDU-032: 教育実施記録は以下の必須項目を含む。記録は本システムの教育記録機能または Excel 添付で本システムに保存する。

| 必須項目 | 内容 |
|---|---|
| 受講者の氏名とユーザー ID | 本システムのユーザー ID で識別する |
| 受講日時 | 教育実施日時（日付・開始時刻・終了時刻） |
| 教材名と版数 | 使用した教材の名称と版数（v1.0.0 形式） |
| テスト結果 | 操作テスト・口頭確認の合否と得点（テストを実施した場合） |
| 教育実施者の氏名と電子署名 | 教育担当者（現場監督・品質担当・IT 担当）の電子署名 |
| 次回更新教育の予定日 | 認定有効期限から逆算した更新教育の予定日 |

EDU-033: 教育実施記録の保存期間は本番稼働後 7 年以上とする。7 年経過後の廃棄は品質担当の承認を得て実施し、廃棄記録を保存する（定期メンテナンス要件 MNT-044 と同様の手順を適用する）。

### 2-2. 教育記録の管理要件

EDU-034: 教育記録の管理は以下の要件を満たす。

| 要件 | 内容 |
|---|---|
| ALCOA+ 準拠 | 新規作成の教育記録は ALCOA+ 準拠の記録として管理する（is_legacy=false） |
| 変更不可 | 完了済み教育記録の内容変更を禁止する（Append-only で管理する） |
| 電子署名 | 教育実施者の電子署名を必須とし、署名なしの記録を無効とする |
| 検索可能性 | 受講者のユーザー ID・受講日・教材版数での検索ができること |

**本節で確定した方針**
- 教育実施記録の 6 必須項目を確定し、7 年以上保存と廃棄時の品質担当承認を確定する。
- 教育記録を ALCOA+ 準拠記録として管理し、電子署名必須・変更禁止を確定する。
- 次回更新教育の予定日を記録必須項目として確定し、スケジュール管理との連動を要件とする。

## 3. Kirkpatrick 4 レベル評価の実施要件

### 3-1. L1（反応）評価

EDU-035: L1（反応）評価は教育完了後 24 時間以内にアプリ内アンケートを実施する。

| 実施要件 | 内容 |
|---|---|
| 実施タイミング | 初期教育完了後・更新教育完了後 |
| アンケート形式 | 1〜5 段階評価（5 が最高）+ 自由記述 |
| 必須質問項目 | (1) 教育内容の理解しやすさ (2) 操作の習得感 (3) 改善の提案（自由記述） |
| 結果の保存 | 本システムの評価記録として保存する |

### 3-2. L2（学習）評価

EDU-036: L2（学習）評価は操作テスト・口頭確認・実技確認で実施する。

| 評価方法 | 実施タイミング | 合格基準 |
|---|---|---|
| 操作テスト（作業員） | 初期教育完了時 | 対象 SOP を自立操作で完走（ミス 0 件） |
| 口頭確認（IT 担当） | 初期構築完了時 | 障害対応 5 フェーズを正確に説明できること |
| 実技確認（IT 担当） | 初期構築完了時 | バックアップ・リストアを手順書のみで完了できること |

### 3-3. L3（行動）・L4（業績）評価

EDU-037: L3（行動）評価は月次の工程改善会議で現場監督が以下を確認する。

| 確認項目 | 評価方法 |
|---|---|
| システム採用率（SLO 計測値） | OPS-050 で定義したシステム採用率の月次値で評価する |
| 不適合起票率の変化 | 不適合発見件数と起票件数の比率（工程・シフト集計）を観察する |
| Kaizen Teian 起票数の変化 | 月次の Kaizen Teian 起票件数（工程集計）を観察する |

EDU-038: L4（業績）評価は月次の工程集計値で評価する。個人別の評価は倫理境界宣言（計画 01 章）に基づき禁止する。

| 評価指標 | 計測方法 |
|---|---|
| 工程別の不適合件数の推移 | 月次の不適合記録件数（工程・シフト別集計）の前月比 |
| SOP 改訂反映時間 | SOP 改訂承認から全端末への同期完了までの時間（SLI-01 関連） |

**本節で確定した方針**
- L1 評価は教育完了後 24 時間以内の 3 質問アンケート・L2 評価は操作テスト・L3・L4 は月次工程集計値の評価方式を確定する。
- L4 評価の個人別評価禁止を倫理境界宣言と連動させて確定する。
- L3・L4 の評価指標として不適合件数推移・SOP 改訂反映時間の 2 指標を確定する。
