# 教育要件サブ

## 1. 本サブの責務

教育要件サブは IPA 共通フレーム 2013 要件定義プロセスの「教育・訓練要件の定義」アクティビティに相当する。

本サブが確定する内容は以下の 3 点に限定する。

- **OJT 設計要件（EDU-OJT）**: 習熟度別の OJT カリキュラム・システム習熟の確認基準
- **多言語教材要件（EDU-ML）**: 外国人技能実習生・特定技能労働者向けの多言語インターフェース・プライバシー通知・操作マニュアルの要件
- **Just Culture 教育要件**: 作業員・現場監督・品質担当への Just Culture 組織原則の教育要件

### 計画書からの継承

本サブは計画書 `09_運用・組織・教育方針.md` の教育設計方針と、構想書 `04_倫理スタンスと支援・監視の意図的分離.md` の Just Culture 採用宣言を教育要件として具体化する。

外国人技能実習生・特定技能労働者（P05 前提条件）への多言語対応は Must 要件として確定する。

---

**本節で確定した方針**
- 教育要件サブは「習熟度別 OJT カリキュラム」「多言語プライバシー通知（日本語・英語必須）」「Just Culture の組織教育」の 3 点を EDU 要件として確定する。
- 多言語教材の提供言語は「日本語・英語」を Must とし、実際の在籍者の母語（ベトナム語・インドネシア語等）を Should とする。
- OJT 完了の受入基準は「初回 OJT 後、追加訓練なしに基本操作を習得できること」を検証可能命題として確定する。
