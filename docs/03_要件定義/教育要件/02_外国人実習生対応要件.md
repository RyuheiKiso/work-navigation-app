# 02 外国人実習生対応要件

本書の責務は、外国人技能実習生・特定技能労働者が本システムを安全に使用できるための言語対応・やさしい日本語・教材多言語化・監督体制の要件を確定することである。上流文書「システム化計画 09 章 4 節（多言語教育の方針）・11 章前提 P05」の方針を受けて、EDU-NNN として要件化する。

---

## 1. 対応言語の要件

### 1-1. 必須サポート言語の確定

EDU-015: 本システムの UI・SOP テキスト・教育教材において日本語と英語の 2 言語を必須サポート言語として確定する。日本語・英語以外の言語は工場の在籍外国人労働者の母語に応じて拡張対応する。

| 言語区分 | 対応言語 | 提供形式 |
|---|---|---|
| 必須 | 日本語 | UI 全体・SOP テキスト・教育教材の全量 |
| 必須 | 英語 | UI 全体・SOP テキスト（対訳）・教育教材（翻訳版） |
| 拡張（工場判断） | ベトナム語・インドネシア語・フィリピン語・中国語・その他 | SOP テキスト（対訳）・操作マニュアル（翻訳版） |

EDU-016: 拡張言語の追加は工場管理者が必要性を判断し、IT 担当に要請する。IT 担当は要請を受けて翻訳対象 SOP・教材のリストを確定し、品質担当が翻訳品質を承認してから本システムに登録する。翻訳品質の承認なしの拡張言語 SOP の有効化を禁止する。

**本節で確定した方針**
- 日本語と英語を必須サポート言語として確定し、拡張言語は工場管理者の判断に基づいて追加する。
- 拡張言語 SOP の有効化には品質担当の翻訳品質承認を必須とする。
- 翻訳品質承認なしの拡張言語 SOP 有効化を技術的に禁止することを要件とする。

## 2. やさしい日本語対応

### 2-1. やさしい日本語の設計原則

EDU-017: 本システムの SOP テキストおよび UI テキストは「やさしい日本語」の設計原則に準拠する。やさしい日本語の基準は以下の通りとする。

| 設計原則 | 具体的な基準 |
|---|---|
| 平易語彙の使用 | 日常生活で頻繁に使用される語彙を優先する。専門用語は漢字表記と平仮名表記（ルビ）を併記する |
| 文の長さ制限 | 1 文あたり 40 文字以内を目安とする。接続詞で繋いだ長文を禁止する |
| 画像・アイコン優先 | テキストのみの説明より、画像・アイコン・色による視覚的説明を優先する |
| 一文一情報 | 1 文に 1 つの情報のみを含める。複数の操作手順を 1 文にまとめることを禁止する |

EDU-018: SOP の Step テキストに対してルビ（振り仮名）付与の要件を以下の通り確定する。

| 対象テキスト | ルビ付与基準 |
|---|---|
| 常用漢字以外の漢字を含む語彙 | ルビ付与を必須とする |
| 技術用語・製造業固有の専門語 | ルビ付与に加え、脚注で平易な説明を付与することを推奨する |
| 数字・単位 | アラビア数字と SI 単位記号（mm・kg・℃ 等）を使用し、漢数字・略称の使用を禁止する |

EDU-019: 平易語彙リストを以下の通り管理する。

| 管理項目 | 内容 |
|---|---|
| リストの形式 | 「原語 → やさしい日本語での表現」のペアで管理する |
| 管理場所 | 本システムの文書管理エリア（Git リポジトリの `docs/glossary/easy_japanese.md`） |
| 更新頻度 | SOP に新規専門用語が登場した場合に品質担当が随時追記する |

**本節で確定した方針**
- やさしい日本語の 4 設計原則（平易語彙・40 文字以内・画像優先・一文一情報）を確定する。
- 常用漢字以外へのルビ付与必須・技術用語への脚注推奨・SI 単位使用を確定する。
- 平易語彙リストを Git リポジトリで管理し、品質担当が随時更新することを確定する。

## 3. 教育教材の多言語化要件

### 3-1. 翻訳対象の確定

EDU-020: 以下の教材を多言語化の対象として確定する。翻訳対象の優先順位は重要度順に設定する。

| 優先順位 | 翻訳対象 | 必須言語 |
|---|---|---|
| P1（最優先） | データ利用ポリシー同意書 | 在籍外国人労働者の全母語 |
| P1 | P1 工程の SOP テキスト（全 Step） | 英語 + 拡張言語 |
| P2 | タブレット APP 操作マニュアル（作業員向け） | 英語 + 拡張言語 |
| P2 | QRC（クイックリファレンスカード） | 英語 + 拡張言語 |
| P3 | P2〜P4 工程の SOP テキスト | 英語 + 拡張言語（必要に応じて） |

EDU-021: 翻訳品質の基準を以下の通り確定する。

| 基準 | 内容 |
|---|---|
| 母語話者レビュー | 翻訳後に当該言語の母語話者がレビューし、意味の正確性を確認する |
| 品質担当の承認 | 母語話者レビュー完了後に品質担当が翻訳版 SOP を承認する（電子署名で承認する） |
| 機械翻訳の禁止 | 機械翻訳のみによる SOP テキストの確定を禁止する。機械翻訳は参考資料として使用し、必ず母語話者が修正する |

EDU-022: SOP の多言語版承認を「新版 SOP の有効化条件」として確定する。外国人労働者が在籍する工場では、母語話者レビュー完了の多言語版を添付しなければ新版 SOP を有効化できない。単一言語版のみでの有効化を禁止する。

**本節で確定した方針**
- 翻訳対象の優先順位（データ利用ポリシー同意書・P1 工程 SOP を最優先）と必須言語を確定する。
- 母語話者レビュー・品質担当承認・機械翻訳禁止の 3 翻訳品質基準を確定する。
- 外国人労働者在籍工場での多言語版承認を新版 SOP 有効化の必須条件として確定する。

## 4. 実習生の監督体制要件

### 4-1. スキル等級ゲートの確定

EDU-023: 外国人技能実習生・特定技能労働者の作業可否は、本システムのスキル認定（スキル等級ゲート）で管理する。

| スキル等級 | 作業可否 | 条件 |
|---|---|---|
| 等級 0（未認定） | 作業不可（本システムでの自立作業不可） | 初期教育未完了・操作テスト未合格 |
| 等級 1（見習い） | 監督者立会いのもとで作業可能 | 初期教育完了・操作テスト合格・同意書署名完了 |
| 等級 2（自立） | 単独作業可能（P1 工程は除く） | 等級 1 取得後 14 日間以上の実績・監督者の自立確認 |
| 等級 3（熟練） | 全工程での単独作業可能 | 等級 2 取得後 30 日間以上の実績・技能検定相当の評価合格 |

EDU-024: P1（最優先）工程は等級 2 以上の作業員のみが担当できる。等級 1 の作業員が P1 工程を担当する場合は現場監督の立会いを必須とする。等級 0 の作業員が P1 工程に関与することを禁止する。

### 4-2. 監督者の立会い基準

EDU-025: 等級 1 の外国人実習生・特定技能労働者が作業を実施する場合の監督者立会い基準を以下の通り確定する。

| 立会い種別 | 適用工程 | 立会い者 |
|---|---|---|
| 同一工程での常時立会い | P1 工程 | 現場監督または等級 3 の先輩作業員 |
| 定期確認立会い | P2 工程 | 監督者が 30 分に 1 回以上確認する |
| 開始時確認のみ | P3・P4 工程 | 作業開始時に監督者が確認する（継続立会い不要） |

EDU-026: 外国人実習生・特定技能労働者のスキル等級変更は現場監督が申請し、品質担当が本システムで確認・承認する。スキル等級の不正変更（自己申告での等級アップ等）を禁止し、承認権限を品質担当に限定する。

**本節で確定した方針**
- スキル等級 0〜3 の 4 段階と各等級の作業可否・条件を確定する。
- P1 工程は等級 2 以上のみ担当可能、等級 1 は立会い必須、等級 0 は P1 工程関与禁止を確定する。
- スキル等級変更の承認権限を品質担当に限定し、現場監督申請・品質担当承認の 2 段階を確定する。
