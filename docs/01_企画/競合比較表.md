## 本アプリの戦略的優位ポイント（5 軸）

「全方位でトップ」は妥協のリスクが高いため、**圧倒的に上回る軸** と **同等を維持する軸** を分離する。

| 戦略 | 軸 | 達成手段 |
| --- | --- | --- |
| 圧倒的優位（◎を狙う） | A4 オフライン対応 | Tauri+SQLite による完全ローカル動作 |
| 圧倒的優位 | A6 対応端末 | Android＋Windows 両系統で同等 UX（iOS 偏重競合への優位） |
| 圧倒的優位 | A7 拡張性・アドオン | 正式 API・サンドボックス・署名 |
| 圧倒的優位 | A8 セットアップ容易性 | `docker compose up` 5 分・端末初回 10 分 |
| 圧倒的優位 | A9 ライセンス・TCO | Apache-2.0 OSS、ライセンス費ゼロ |
| 圧倒的優位 | A11 規制適合 | GxP／IATF／ISO 13485 のオプション機能（電子記録・監査証跡・電子署名） |
| 同等以上を維持 | A1 ナビ粒度 | §2 ドメインモデルで Tulip／VKS と同水準を確保 |
| 同等以上を維持 | A2 非エンジニア操作性 | 設定 UI で SwipeGuide／Poka 水準 |
| 同等以上を維持 | A3 ERP/MES 連携 | REST 双方向、OPC UA はアドオン |
| 同等以上を維持 | A5 メディア対応 | カメラ・動画・音声・画像 |
| 同等以上を維持 | A10 国際化 | 日英 2 ロケール標準、翻訳ファイル追加方式 |

「劣る」（△・×）は本アプリでは許容しない。検出時はロードマップ §22 サイクルで是正対象。
