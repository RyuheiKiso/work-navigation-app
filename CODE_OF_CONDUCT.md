# Contributor Covenant 行動規範

> 対応 §: ロードマップ §19.2 §19.4
> 採用バージョン: Contributor Covenant v2.1
> 違反対応 SLA: 7 日以内に着手（§19.4.4）

## 私たちの誓約

メンバー、コントリビューター、リーダーである私たちは、年齢、体格、目に見える／見えない障がい、民族性、性自認・性表現、経験、教育、社会経済的地位、国籍、容姿、人種、カースト、宗教、性的指向にかかわらず、誰もがハラスメントのないコミュニティに参加できることを誓います。

私たちはオープンで歓迎的かつ多様で、包摂的で健全なコミュニティに寄与する形で行動し、相互に交流することを誓います。

## 私たちの基準

私たちのコミュニティに対し、肯定的な環境を作り出すことに寄与する行動の例:

- 他者への共感と思いやりを示す
- 異なる意見、視点、経験を尊重する
- 建設的なフィードバックを行い、丁寧に受け入れる
- 私たちの行動の影響に責任を持ち、影響を受けた人々に謝罪し、その経験から学ぶ
- 個々人だけでなく、コミュニティ全体にとって最善のことに焦点を当てる

容認できない行動の例:

- 性的な言葉・画像の使用、いかなる種類の性的注目・性的なアプローチ
- トローリング、侮辱・軽蔑的なコメント、個人攻撃・政治的攻撃
- 公的または私的な嫌がらせ
- 物理的・電子メールアドレス等の他者の個人情報を、明示的な許可なく公開すること
- 専門的な場において合理的に不適切と見なされうる、その他の行動

## 執行責任

コミュニティのリーダーは、容認できる行動の基準を明確化・執行する責任があり、不適切・脅威的・攻撃的・有害だと判断した行動に対して、適切かつ公正な是正措置を取ります。

コミュニティのリーダーは、本行動規範に整合しないコメント、コミット、コード、wiki 編集、Issue、その他の貢献を削除・編集・拒否する権限を持ち、適切な場合はモデレーション判断の理由を伝えます。

## 適用範囲

本行動規範はすべてのコミュニティ空間に適用され、また、個人がコミュニティを公的に代表している場合の公的な場にも適用されます。代表行為の例には、公式メールアドレスの使用、公式 SNS アカウントを通じた投稿、オンライン／オフラインイベントでの代理人としての行動が含まれます。

## 執行手順

- 罵声・嫌がらせ・その他容認できない行動の事例は、メンテナへの GitHub Security Advisories または `MAINTAINERS.md` 記載の連絡経路で報告してください。
- すべての苦情は迅速かつ公正にレビュー・調査されます。
- すべてのコミュニティリーダーは、報告者のプライバシーと安全を尊重する義務があります。
- 違反対応 SLA は **7 日以内に着手**（§19.4.4）。

## 執行ガイドライン

コミュニティのリーダーは、本行動規範違反の結果と判断する際、次のコミュニティ影響ガイドラインに従います。

### 1. 是正

**コミュニティ影響**: 不適切な言葉、または専門的でない、または歓迎されない行動。

**結果**: 違反の性質を明確化し、なぜその行動が不適切だったかの説明を含む、コミュニティリーダーからの非公開の書面による警告。公的な謝罪が要求される場合があります。

### 2. 警告

**コミュニティ影響**: 単発、または一連の行動を通じた違反。

**結果**: 継続行動への結果を伴う警告。一定期間、関係者との交流（行動規範執行者との求められざる交流を含む）を禁じる。これにはコミュニティ空間および SNS 等の外部チャネルでの交流回避も含まれる。これらの条件違反は一時的または永続的な禁止につながる場合がある。

### 3. 一時的な禁止

**コミュニティ影響**: 持続的な不適切行動を含む、コミュニティ基準に対する深刻な違反。

**結果**: 一定期間、コミュニティとの公的な交流または公的な対話の一切を禁止する。この期間中は、行動規範執行者との求められざる交流を含む、関係者との公的・私的な交流は許可されない。これらの条件違反は永続的禁止につながる場合がある。

### 4. 永続的な禁止

**コミュニティ影響**: コミュニティ基準への違反のパターンの示唆、個人への持続的嫌がらせ、または個人クラスへの侵害・敵意。

**結果**: コミュニティ内での公的な交流の永続的禁止。
