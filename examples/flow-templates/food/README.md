# 食品 業界フローテンプレート

> 対応 §: ロードマップ §10.2.1 §3.1.5.4 §12（HACCP／ISO 22000）
> 対象規格: HACCP CCP 監視／ISO 22000 食品安全マネジメント
> ライセンス: Apache-2.0

## 概要

HACCP（Hazard Analysis and Critical Control Points）に基づく食品製造フロー。
CCP（Critical Control Point）監視と温度・時間記録を §3.1.1 完了条件で表現する。

## 同梱ファイル

| ファイル | 用途 |
| --- | --- |
| `cooking-ccp.yaml` | 加熱工程（CCP）の温度・時間記録 |
| `cold-chain.yaml` | 冷蔵保管の温度連続監視 |

## 業界語感（§3.1.5.4 整合）

- 「作業」= HACCP プラン上のステップ
- 「工程」= 受入／加工／加熱／冷却／包装／保管／出荷
- 「手順」= レシピに沿った各操作
- 「動作」= 計量・撹拌・温度確認

## CCP 監視

CCP は本アプリの `critical_control_point: true` フラグで表現する。
完了条件には以下を必ず含める:

- 温度記録（温度計連携アドオン経由）
- 時間記録（標準時間との比較）
- 担当者署名（電子署名）
