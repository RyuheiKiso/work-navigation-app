- android
- iOS
- windows

## 技術スタック

### ハンディAPP

React Native
(SQLite + TypeORM をローカルストレージとして使用)

### マスタメンテナンス

React

### バックエンド

Rust(Edition 2024) + tokio + axum

### データベース

PostgreSQL

### デプロイ

IIS + Windows Server 2022
WSL2 + Docker
