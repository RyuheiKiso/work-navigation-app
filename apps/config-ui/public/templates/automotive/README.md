# 自動車部品 業界フローテンプレート

> 対応 §: ロードマップ §10.2.1 §3.1.5.4 §12（IATF 16949） §27（FMEA）
> 対象規格: IATF 16949 §8.3／§8.5.1 標準作業／AIAG-VDA FMEA Handbook 1st Ed. (2019)
> ライセンス: Apache-2.0（本リポジトリと同一）

## 概要

自動車部品の組立ラインを想定した最小フローテンプレート。
タクトに刻まれた標準作業（IATF 16949 §8.5.1）をベースに、
完了条件・標準時間・トルク検査などを §3.1.1 ドメインモデルに沿って表現する。

## 同梱ファイル

| ファイル | 用途 |
| --- | --- |
| `assembly-line.yaml` | 組立ライン基本フロー（4 工程） |
| `setup-changeover-smed.yaml` | 段取り替え（SMED）の内段取／外段取分離フロー |
| `quality-hold.yaml` | 品質ホールド検出時の例外フロー |

## 業界語感（§3.1.5.4 整合）

- 「作業」= タクトに刻まれた **標準作業**（standard work）
- 「工程」= ステーション（station）
- 「手順」= ピックアップ／組付／検査／計測 の各動作系列
- 「動作」= 把持・回転・締結 の最小単位

## FMEA 例（§27 ／ `examples/fmea-iatf16949/`）

| ID | 故障モード | 本テンプレでの予防 |
| --- | --- | --- |
| AT-001 | 規格外部材の使用 | `receive_parts` の `completion_criteria: photo` で目視＋撮影 |
| AT-002 | トルク不足 | `torque_check` ステップを必須化／OPC UA アドオンで実測値連携 |
| AT-003 | 検査記録の漏れ | `quality-hold.yaml` で 4 ロール RBAC・電子署名 |
| AT-004 | 工程順序逆転 | フロー定義の HSM／CPN 形式検証（§3.4.1） |

## 受入観点

- 設定 UI（§7.2）から本ディレクトリの YAML を「インポート」操作で読み込めること。
- 5 階層ネスト（工場・ライン・セル・工程・手順）の最低 4 階層を表現すること。
- §27 FMEA 行と本テンプレの対応関係が `examples/fmea-iatf16949/README.md` と双方向参照可能であること。
