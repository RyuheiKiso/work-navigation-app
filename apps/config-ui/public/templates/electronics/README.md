# 電子部品 業界フローテンプレート

> 対応 §: ロードマップ §10.2.1 §3.1.5.4 §6.3 §12（ISO 9001／IPC）
> 対象規格: ISO 9001 品質管理／IPC-A-610 はんだ付け基準
> ライセンス: Apache-2.0

## 概要

クリーンルーム制約下の電子部品製造フロー。
半導体・電子組立の精密手順、はんだ付け検査、ESD（静電気対策）を §3.1.1 完了条件で表現する。

## 同梱ファイル

| ファイル | 用途 |
| --- | --- |
| `smt-assembly.yaml` | SMT（表面実装）組立フロー |
| `cleanroom-entry.yaml` | クリーンルーム入退室手順 |

## 業界語感（§3.1.5.4 整合）

- 「作業」= 精密手順（precision procedure）
- 「工程」= SMT／ICT／FCT／パッケージング
- 「手順」= はんだ付け／検査／クリーニング
- 「動作」= ピックアップ／配置／検査

## クリーンルーム特有の対応

- 手袋必須（§6.3 推奨機種は Zebra ET60 等の手袋対応端末）
- ESD 防止帯チェック
- 顔認証アドオン（§10.5.0 `auth.extend:face`）でマスク着用下の本人確認
